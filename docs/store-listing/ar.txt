PlainTab هي صفحة تبويب جديدة جميلة وسريعة وهادئة.

لماذا؟ لأن صفحة التبويب الجديدة يجب أن تعرض لك أولًا خلفية مريحة، لا إعلانات أو أخبار أو قوائم رائجة أو شاشة مليئة بالاختصارات.

يقوم PlainTab بشيء بسيط: تفتح تبويبًا جديدًا، ترى الخلفية، ثم تبحث أو تكتب عنوان URL أو تفتح اختصاراتك الخاصة. لا ينافس على انتباهك، ولا يحول الصفحة الرئيسية للمتصفح إلى موجز محتوى آخر.

لماذا تختار PlainTab؟

【صورة تظهر فورًا】
• تحميل محلي فائق السرعة للخلفيات: يحفظ PlainTab لقطة معاينة خفيفة محليًا ويحمّلها فورًا عند فتح كل تبويب جديد، مما يقلل وميض الشاشة البيضاء.
• تنتقل الخلفيات عالية الدقة بسلاسة في الخلفية، بينما تبقى الصورة الحالية كطبقة احتياطية في الأسفل، لذلك لا تحتاج إلى انتظار الشبكة لرؤية الخلفية.
• تدعم خلفيات Bing اليومية التخزين المؤقت والتحديث؛ بعد استخدام PlainTab مرة واحدة تصبح الفتحات اللاحقة أكثر سلاسة.

【حرية الخلفيات】
• يدعم خلفيات Bing اليومية وصورك التي ترفعها بنفسك.
• يدعم اختيار مجلد خلفيات محلي، وهو مناسب إذا كانت لديك مجموعة خلفيات جاهزة.
• يدعم مصادر صور RSS وواجهات API مخصصة: بسيط عندما تريد البساطة، ومرن عندما تريد التخصيص.
• تدعم الخلفيات المحلية تدوير ما يصل إلى 12 صورة مع معاينة معرض 3×4، ويمكن حذفها أو استبدالها في أي وقت.

【هادئ لكنه غير فارغ】
• تبقى الصفحة الرئيسية بسيطة: خلفية، شريط بحث، وبعض المداخل الضرورية، من دون ازدحام بالودجت.
• يمكن ضبط موضع شريط البحث وحجمه وزواياه وشفافيته وطريقة ظهوره.
• يمكن أن يبقى شريط البحث ظاهرًا دائمًا، أو يظهر عند الحاجة فقط، أو يبقى مخفيًا.
• تتم إدارة الاختصارات في لوحة الأوامر، مع دعم البحث والإضافة والتعديل واستيراد الإشارات المرجعية، من دون تغطية الخلفية.
• تتيح المساحة المخفية حفظ الروابط التي لا تريد عرضها مباشرة، مع الحفاظ على نظافة الصفحة.

【الخصوصية أولًا】
• لا إعلانات، لا تتبع، ولا جمع لبيانات التصفح.
• يتم حفظ الخلفيات المرفوعة محليًا في تخزين المتصفح المحلي. لا يتم رفعها أو مزامنتها أو تمريرها عبر خوادم خارجية.
• إعدادات البحث والاختصارات مصممة للاستخدام المحلي. يجب أن تكون صفحة التبويب الجديدة ملكًا لك.

【خفيف وشفاف】
• مبني باستخدام JavaScript أصلي و CSS وواجهات المتصفح، بلا إطار عمل للواجهة، بلا خطوة بناء، وبلا اعتماديات تشغيل كبيرة.
• يعمل كإضافة تبويب جديد في Chrome / Edge، ويمكن فتحه مباشرة كصفحة ويب أيضًا.
• مفتوح المصدر بالكامل، مناسب للاستخدام اليومي ولتعلم تطوير إضافات المتصفح ومشاريع الواجهة الأمامية والهندسة بمساعدة الذكاء الاصطناعي.

PlainTab مناسب لك إذا كنت تريد:

• تبويبًا جديدًا أجمل من دون ضجيج موجز المعلومات
• خلفيات يومية أو مكانًا لإدارة مجموعة خلفياتك الخاصة
• البحث والاختصارات واستيراد الإشارات المرجعية والنسخ الاحتياطي من دون ازدحام الصفحة
• صفحة رئيسية لا تغطيها عشرات مربعات الاختصارات
• إضافة خفيفة وشفافة من دون أدوات بناء معقدة
• مشروع إضافة متصفح حقيقي وكامل للتعلم منه

ما الذي لا يفعله PlainTab:

• لا موجزات أخبار أو قوائم رائجة أو محتوى موصى به
• لا إعلانات أو بطاقات ممولة أو مساحات ترويجية
• لا نظام حسابات أو ميزات اجتماعية أو تدفق محتوى سحابي
• لا جدار من الاختصارات يغطي الصفحة الرئيسية
• لا فيديوهات تشغيل تلقائي أو خلفيات متحركة

سجل التحديثات

• v3.2.0 · تحديث كبير للميزات والأداء: إضافة لوحة الأوامر، خلفيات RSS/API/المجلد المحلي، ألوان سمات خلفية WASM، استيراد/تصدير الإعدادات وسجل البحث؛ ترحيل التخزين إلى نموذج v3.2، تشديد الأذونات واحتياطي البحث، وتحسين مسارات البدء والخلفيات والمعرض والإعدادات عبر تقسيمها إلى وحدات.

• v3.1.4 · إعادة تسمية ثوابت التخزين ودمج ترحيل IDB إلى معاملة ثنائية المرحلة لترتيب قابل للاسترداد بعد الأعطال؛ تحسين تخطي الترحيل للمستخدمين الجدد؛ إضافة إزالة التكرار في الرفع الجماعي مع حد 12 صورة

• v3.1.3 · تنفيذ ترحيل إصدار التخزين لـ localStorage و IndexedDB، توحيد تسمية مفاتيح التخزين، إصلاح منطق استمرار الصور المصغرة في الوضع المحلي.

• v3.1.2 · إعادة تسمية _locales/pt إلى pt_BR للامتثال لمتجر Chrome Web Store؛ توحيد تنسيق changelog؛ إعادة تسمية ملفات البرتغالية إلى pt_BR.

• v3.1.1 · تحويل Bing API إلى وضع سباق Promise.any مع مهلة AbortController لمدة 8 ثوانٍ؛ تحديث ملفات README بكل اللغات الـ 16.

• v3.1.0 · معرض الخلفيات المحلية: رفع وحذف وتدوير عدة خلفيات، اختيار جماعي للملفات؛ تحسين تنسيق اللوحات؛ تعزيز الترجمة؛ تشديد إعدادات CSP.

• v3.0.5 · ترقية وكيل Bing API، تحسين تخزين الخلفيات المؤقت، توحيد مفاتيح localStorage، إضافة changelog بـ 16 لغة، تحسين السجلات ودلالات الكود.

• v3.0.4 · إصلاح فشل تحميل الخلفيات بسبب نقص نطاقات Bing CDN في قائمة السماح للصور في CSP.

• v3.0.3 · تحسين مسار الإطار الأول، تعزيز أمان CSP، تنظيف الكود غير المستخدم، إضافة أيقونات ومواد ترويجية.

• v3.0.2 · خلفية متكيفة مع الوضع الداكن، تعديل بنية الكود، إضافة شريط سفلي متعدد اللغات.

• v3.0.1 · دمج Chrome Search API لتلبية متطلبات الغرض الواحد في Chrome Web Store.

• v3.0.0 · إعادة كتابة النواة: بنية مزدوجة الطبقات تقلل وميض الشاشة البيضاء في التبويبات الجديدة، مع خلفية احتياطية.

• v2.2.7 · إضافة README بالإنجليزية واليابانية والكورية والروسية والصينية المبسطة؛ تغيير وضع البحث الافتراضي إلى ظاهر دائمًا.

• v2.2.6 · تحسين تحميل الخلفيات وتخزين الصور المصغرة، وتنظيم ثوابت Bing API.

• v2.2.5 · إضافة تحميل مسبق للخلفيات وتحسين الإطار الأول لتقليل الشاشة البيضاء.

• v2.2.4 · ترحيل تخزين الخلفيات من localStorage إلى IndexedDB.

• v2.2.3 · إضافة رفع الخلفيات المحلية وتحسين تجربة تبديل محرك البحث.

• v1.2.0 · الإصدار الأول: نظام خلفيات، واجهة بـ 16 لغة، توطين Chrome i18n، ورخصة MIT.
