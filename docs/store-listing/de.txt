PlainTab ist eine Browsererweiterung, die jeden neuen Tab zu einem optischen Genuss macht.

Warum? Weil Ihr neuer Tab ein schönes Hintergrundbild verdient — keine Werbung, keine Nachrichten, keine Schnellzugriffe.

Warum PlainTab?

• 【Wandbildfreiheit】
• Tägliche automatische Aktualisierung mit dem Bing-Bild des Tages — zwei globale Endpunkte im Wettlauf, blitzschnell egal wo Sie sind.
• Extrem schnelle lokale Wandbilder — die Bilder werden auf Ihrem Gerät gespeichert und beim Öffnen eines neuen Tabs sofort geladen, eine Größenordnung schneller als Netzwerkzugriffe.
• Unterstützt bis zu 12 lokale Wandbilder mit automatischer Rotation bei jedem neuen Tab. 3×4 Galerieansicht, jederzeit löschbar und austauschbar.

• 【Absoluter Datenschutz】
• Keine Werbung, kein Tracking, keine Datensammlung.
• Alle Wandbilder werden offline auf Ihrem Gerät gespeichert — kein Hochladen, keine Synchronisation, keine Übertragung über Server. Ihr Surfverhalten gehört nur Ihnen.

• 【Blitzschnell, unaufdringlich】
• Reines natives JavaScript, null externe Abhängigkeiten. Kein Drittanbieter-Framework — beim Öffnen sofort bereit.
• Wandbilder erstellen im Hintergrund eine extrem leichte Vorschauminiatur. Beim nächsten neuen Tab wird sie direkt vom Gerät geladen — ohne Netzwerk.

• 【Flexibel, nicht kompliziert】
• Automatische Spracherkennung aus 16 Sprachen.
• Suchleiste: bei Mauszeiger-Kontakt einblenden / immer anzeigen / ausblenden.
• Suchmaschine: Google / Bing / Baidu / DuckDuckGo (im Web-Modus).

• 【Open Source, transparent】
• Vollständig quelloffen: https://github.com/kaininx/PlainTab
• Online-Demo: https://plaintab.kaininx.workers.dev

✨ Empfohlene Kombination (vom Entwickler):
• 📌 Neuer Tab: Installieren Sie die Erweiterung und legen Sie bis zu 12 verschiedene Wandbilder beliebiger Größe fest.
• 📌 Startseite: Setzen Sie die Startseite des Browsers auf https://plaintab.kaininx.workers.dev , um den täglichen Bing-Modus zu nutzen.
• 📌 Home-Button: Setzen Sie ihn auf https://kaininx.github.io/PlainTab/ — das GitHub Pages dieser Erweiterung, mit zusätzlichen eigenen Wandbildern.

PlainTab ist genau richtig, wenn Sie genug haben von:
• Werbung, Nachrichten und nutzlosen Schnellzugriffen auf dem neuen Tab
• Browsererweiterungen, die im Hintergrund heimlich Ihre Daten sammeln
• trägem Browser durch zu viele Erweiterungen — Sie wollen native Geschwindigkeit
• dem immer gleichen Bild auf jedem neuen Tab
