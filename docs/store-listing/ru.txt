PlainTab — красивая, быстрая и спокойная страница новой вкладки.

Почему? Потому что новая вкладка должна сначала показывать приятные обои, а не рекламу, новости, тренды или экран, заполненный ярлыками.

PlainTab делает одну простую вещь: вы открываете новую вкладку, видите обои, а затем можете искать, вводить URL или открывать свои ссылки. Он не борется за ваше внимание и не превращает домашнюю страницу браузера в ещё одну ленту.

Почему PlainTab?

【Изображение сразу】
• Очень быстрая локальная загрузка обоев: PlainTab сохраняет лёгкий предпросмотр локально и мгновенно загружает его при каждом открытии новой вкладки, уменьшая белые вспышки.
• Обои высокого разрешения плавно меняются в фоне, а предыдущая картинка остаётся нижним резервным слоем. Вам не нужно ждать сеть, чтобы увидеть фон.
• Ежедневные обои Bing поддерживают кэширование и обновление; после первого использования последующие открытия становятся плавнее.

【Свобода обоев】
• Поддерживаются ежедневные обои Bing и ваши собственные загруженные изображения.
• Можно выбрать локальную папку с обоями — удобно, если у вас уже есть коллекция.
• Поддерживаются RSS-источники изображений и пользовательские API: просто, если нужно просто, и гибко, если хочется настроить больше.
• Локальные обои поддерживают ротацию до 12 изображений и предпросмотр галереи 3×4, их можно удалить или заменить в любой момент.

【Спокойно, но не пусто】
• Главная страница остаётся сдержанной: обои, строка поиска и несколько нужных элементов, без перегруза виджетами.
• Строку поиска можно настраивать по положению, размеру, скруглению, прозрачности и режиму отображения.
• Она может быть всегда видимой, появляться только при необходимости или оставаться скрытой.
• Ярлыки управляются через командную панель: поиск, добавление, редактирование и импорт закладок, без перекрытия обоев.
• Скрытое пространство позволяет хранить ссылки, которые не хочется показывать напрямую, сохраняя главную страницу чистой.

【Приватность прежде всего】
• Без рекламы, без трекинга, без сбора данных о просмотре.
• Локально загруженные обои хранятся в локальном хранилище браузера. Они не загружаются, не синхронизируются и не проходят через сторонние серверы.
• Настройки поиска и ярлыков рассчитаны на локальное использование. Новая вкладка должна принадлежать вам.

【Лёгкий и прозрачный】
• Создан на чистом JavaScript, CSS и браузерных API, без frontend-фреймворка, без шага сборки и без крупных runtime-зависимостей.
• Работает как расширение новой вкладки для Chrome / Edge, а также может открываться напрямую как веб-страница.
• Полностью open source, подходит для ежедневного использования и для изучения расширений браузера, frontend-проектов и разработки с участием ИИ.

PlainTab подойдёт вам, если вы хотите:

• Более красивую новую вкладку без шума информационных лент
• Ежедневные обои или место для управления собственной коллекцией
• Поиск, ярлыки, импорт закладок и резервное копирование без захламления страницы
• Домашнюю страницу, не покрытую десятками плиток ярлыков
• Лёгкое и прозрачное расширение без сложных инструментов сборки
• Реальный и полноценный проект расширения браузера для обучения

Чего PlainTab не делает:

• Нет новостных лент, трендов или рекомендаций
• Нет рекламы, спонсорских карточек или промо-блоков
• Нет аккаунтов, социальных функций или облачной ленты контента
• Нет стены ярлыков, закрывающей главную страницу
• Нет автозапуска видео или анимированных фонов

История версий

• v3.2.0 · Крупное обновление функций и производительности: добавлены командная панель, обои из RSS/API/локальной папки, WASM-цвета темы обоев, импорт/экспорт настроек и история поиска; хранилище перенесено на модель v3.2, усилены разрешения и поисковый fallback, модульно оптимизированы запуск, обои, галерея и настройки.

• v3.1.4 · Переработано именование констант хранилища и объединена миграция IDB в двухфазную транзакцию с восстановлением после сбоев; оптимизирован пропуск миграции для новых пользователей; добавлена дедупликация при пакетной загрузке с лимитом 12 изображений

• v3.1.3 · Реализована миграция версий хранилища для localStorage и IndexedDB, унифицированы имена ключей, исправлена логика сохранения миниатюр в локальном режиме.

• v3.1.2 · _locales/pt переименован в pt_BR для соответствия Chrome Web Store; формат changelog унифицирован; португальские файлы переименованы в pt_BR.

• v3.1.1 · Bing API переведён в режим гонки Promise.any с 8-секундным таймаутом AbortController; обновлены README на 16 языках.

• v3.1.0 · Галерея локальных обоев: загрузка, удаление и ротация нескольких обоев, пакетный выбор; улучшен стиль панелей; расширена локализация; ужесточена CSP-конфигурация.

• v3.0.5 · Обновлён прокси Bing API, оптимизирован кэш обоев, унифицированы ключи localStorage, добавлены changelog на 16 языках, улучшены логи и семантика кода.

• v3.0.4 · Исправлена ошибка загрузки обоев из-за отсутствующих доменов Bing CDN в CSP allowlist для изображений.

• v3.0.3 · Оптимизирован путь первого рендера, усилена безопасность CSP, удалён неиспользуемый код, добавлены иконки и промоматериалы.

• v3.0.1 · Интегрирован Chrome Search API для соответствия требованиям единственного назначения Chrome Web Store.

• v3.0.0 · Переписано ядро: двухслойная архитектура уменьшает белые вспышки на новой вкладке, добавлен резервный фон.
