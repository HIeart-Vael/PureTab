PlainTab — это расширение для браузера, которое делает каждую новую вкладку настоящим удовольствием для глаз.

Почему? Потому что ваша новая вкладка заслуживает красивую картинку, а не рекламу, новости и ссылки на быстрый доступ.

Почему PlainTab?

• 【Свобода выбора обоев】
• Ежедневное автоматическое обновление обоев Bing — два глобальных сервера работают одновременно, чтобы обеспечить максимальную скорость, где бы вы ни находились.
• Молниеносная загрузка локальных обоев — изображения хранятся на вашем устройстве и загружаются мгновенно при открытии вкладки, на порядок быстрее сетевых запросов.
• Поддержка до 12 локальных изображений с автоматической сменой при каждом открытии вкладки. Галерея 3×4 для просмотра, удаления и замены в любой момент.

• 【Абсолютная конфиденциальность】
• Без рекламы, без отслеживания, без сбора данных.
• Все обои хранятся локально на вашем устройстве — никаких загрузок, синхронизации или передачи через сторонние серверы. Ваша история просмотра принадлежит только вам.

• 【Мгновенно, без задержек】
• Чистый нативный JavaScript, нулевая зависимость от внешних библиотек. Никаких сторонних фреймворков — страница открывается сразу.
• Обои автоматически создают сверхлёгкую превью-копию в фоне, и при следующем открытии вкладка загружается мгновенно из локального хранилища, без обращений к сети.

• 【Гибко и просто】
• Автоматическое определение языка из 16 доступных.
• Строка поиска: показывать при наведении / всегда показывать / скрыть.
• Поисковая система: Google / Bing / Baidu / DuckDuckGo (в веб-режиме).

• 【Открытый исходный код】
• Полный исходный код: https://github.com/kaininx/PlainTab
• Онлайн-демо: https://plaintab.kaininx.workers.dev

✨ Рекомендуемая комбинация от разработчика:
• 📌 Новая вкладка: установите расширение, добавьте до 12 различных обоев без ограничения по размеру.
• 📌 Стартовая страница: установите в браузере https://plaintab.kaininx.workers.dev , используйте ежедневные обои Bing.
• 📌 Кнопка «Домой»: установите https://kaininx.github.io/PlainTab/ — это GitHub Pages проекта, где можно задать собственные обои.

PlainTab для вас, если вы устали от:
• рекламы, новостей и бесполезных ссылок на странице новой вкладки
• расширений, тайно собирающих ваши данные в фоне
• тормозов браузера из-за обилия расширений — хочется родной скорости
• одной и той же картинки на новой вкладке каждый день

История версий

• v3.1.4 · Переработано именование констант хранилища и объединена миграция IDB в двухфазную транзакцию с восстановлением после сбоев; оптимизирован пропуск миграции для новых пользователей; добавлена дедупликация при пакетной загрузке с лимитом 12 изображений

• v3.1.3 · Реализована миграция версий хранилища для localStorage и IndexedDB, унифицированы имена ключей, исправлена логика сохранения миниатюр в локальном режиме.

• v3.1.2 · Переименование _locales/pt в pt_BR для соответствия требованиям Chrome Web Store; унификация формата changelog в компактный однострочный стиль; переименование португальских файлов в pt_BR для согласованности.

• v3.1.1 · Переключение Bing API на конкурентный режим Promise.any с AbortController-таймаутом 8 секунд; обновлены README на всех 16 языках с описанием новой стратегии загрузки.

• v3.1.0 · Локальная галерея обоев: загрузка, удаление и смена нескольких изображений, пакетный выбор файлов; рефакторинг кода; улучшение стилей панелей; расширение локализации; исправление CSP — удалены домены JSON API.

• v3.0.5 · Обновление прокси-сервера Bing API (регионализированные запросы + дедупликация изображений); оптимизация кэша обоев; унификация ключей localStorage; добавлены changelog на 16 языках; улучшение журналирования и семантики кода.

• v3.0.4 · Исправление ошибки загрузки обоев: CSP img-src не содержал домены CDN Bing, из-за чего перенаправленные изображения блокировались браузером.

• v3.0.3 · Ускорение отрисовки первого кадра через CSSOM, усиление безопасности CSP, удаление мёртвого кода, добавлены SVG-иконки и промо-материалы.

• v3.0.2 · Адаптивный фон для тёмной темы, рефакторинг структуры кода, нижняя панель с выбором языка.

• v3.0.1 · Интеграция Chrome Search API, прохождение проверки Chrome Web Store на единственную цель.

• v3.0.0 · Полная переработка: двухслойная архитектура устраняет белый экран при открытии вкладки, добавлен серый резервный фон.

• v2.2.7 · Добавлены README на английском, японском, корейском, русском и упрощённом китайском; режим поиска по умолчанию изменён на «всегда показывать».

• v2.2.6 · Оптимизация логики загрузки обоев и кэша миниатюр, упорядочивание констант Bing API.

• v2.2.5 · Предзагрузка обоев и оптимизация первого кадра, устранение белого экрана новой вкладки.

• v2.2.4 · Перенос хранения обоев из localStorage в IndexedDB.

• v2.2.3 · Загрузка локальных обоев, улучшение переключения поисковых систем.

• v1.2.0 · Первый релиз: система обоев, интерфейс на 16 языках, локализация Chrome i18n, лицензия MIT.
