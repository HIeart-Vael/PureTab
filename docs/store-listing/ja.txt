PlainTab は、新しいタブを開くたびに目を楽しませてくれるブラウザ拡張機能です。

なぜ？新しいタブページには、広告やニュース、ショートカットリンクではなく、素敵な壁紙がふさわしいからです。

PlainTab を選ぶ理由：

• 【壁紙の自由】
• Bing の美しい写真を毎日自動更新 — 世界中の 2 つのエンドポイントを Promise.any で競速し、どこにいても高速に取得。
• 圧倒的に速いローカル壁紙読み込み — 壁紙はあなたの端末に保存され、新しいタブを開くと即座に読み込まれます。ネットワーク経由より桁違いに高速です。
• 最大 12 枚のローカル壁紙をアップロード可能。タブを開くたびに自動でローテーション。3×4 ギャラリーでいつでも削除・変更可能。

• 【完全プライバシー保護】
• 広告なし、トラッキングなし、いかなるデータも収集しません。
• すべての壁紙はオフラインで端末に保存 — アップロードも同期もサーバー経由も一切なし。あなたのブラウジング行動はあなただけのものです。

• 【超高速、邪魔しない】
• 純粋なネイティブ JavaScript、外部依存ゼロ。サードパーティフレームワーク一切不要。ブラウザを開けば即座に表示。
• 壁紙はバックグラウンドで超軽量プレビュースナップショットを生成。次に新しいタブを開くときは、ローカルから瞬時に読み込まれ、ネットワークを経由しません。

• 【柔軟でシンプル】
• 16 言語に自動対応。
• 検索バー：マウスホバーで表示 / 常に表示 / 非表示。
• 検索エンジン：Google / Bing / Baidu / DuckDuckGo（Web モード）。

• 【オープンソースで透明】
• コードは完全にオープンソース：https://github.com/kaininx/PlainTab
• オンラインで試す：https://plaintab.netlify.app

✨ 組み合わせて使う（開発者おすすめ）：
• 📌 新しいタブ：本拡張機能をインストール。最大 12 枚の異なる壁紙を設定可能、サイズ制限なし。
• 📌 起動ページ：ブラウザのホームページを https://plaintab.netlify.app に設定。デフォルトの Bing 日替わり壁紙モードを維持。
• 📌 ホームボタン：https://kaininx.github.io/PlainTab/ に設定。これは本プロジェクトの GitHub Pages で、お好みの壁紙を追加で設定可能。

PlainTab はこんな方におすすめ：
• 新しいタブに広告やニュース、不要なショートカットが詰め込まれているのにうんざりしている
• ブラウザ拡張機能がバックグラウンドでこっそりデータを収集しているのが気になる
• 拡張機能が多すぎてブラウザが重くなるのが嫌で、ネイティブのような速度を求めている
• 毎日同じ壁紙で飽きている
