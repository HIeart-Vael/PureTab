PlainTab は、美しく、速く、静かな新しいタブページです。

なぜなら、新しいタブにはまず心地よい壁紙が表示されるべきで、広告、ニュース、トレンド、画面いっぱいのショートカットが表示されるべきではないからです。

PlainTab がすることはシンプルです。新しいタブを開くと壁紙が表示され、そのあと検索したり、URL を入力したり、自分のショートカットを開いたりできます。注意を奪わず、ブラウザのホームを別の情報フィードに変えません。

PlainTab を選ぶ理由

【開いた瞬間に画面がある】
• 非常に高速なローカル壁紙読み込み：PlainTab は軽量なプレビュー画像をローカルに保存し、新しいタブを開くたびに即座に読み込むことで、白い画面のちらつきを減らします。
• 高解像度の壁紙はバックグラウンドで滑らかに切り替わり、既存の画像は下層にフォールバックとして残ります。背景を見るためにネットワークを待つ必要はありません。
• Bing 日替わり壁紙はキャッシュと更新に対応しています。一度使ったあとは、次回以降の表示がよりスムーズになります。

【壁紙の自由】
• Bing 日替わり壁紙と、自分でアップロードした画像に対応。
• ローカルの壁紙フォルダを選択できます。すでに壁紙コレクションを持っている人に向いています。
• RSS 画像ソースとカスタム API に対応。シンプルに使うことも、こだわって設定することもできます。
• ローカル壁紙は最大 12 枚のローテーションに対応し、3×4 ギャラリープレビューでいつでも削除・入れ替えできます。

【静かだが、空っぽではない】
• ホーム画面は控えめです。壁紙、検索バー、必要な入口だけで、ウィジェットで埋め尽くしません。
• 検索バーは位置、サイズ、角丸、透明度、表示方式を調整できます。
• 常に表示、必要なときだけ表示、非表示から選べます。
• ショートカットはコマンドパレットで管理します。検索、追加、編集、ブックマークのインポートに対応し、壁紙を覆いません。
• 隠しスペースには直接表示したくないリンクを保存でき、ホーム画面をきれいに保てます。

【プライバシー優先】
• 広告なし、トラッキングなし、閲覧データの収集なし。
• ローカルにアップロードした壁紙はブラウザのローカルストレージに保存されます。アップロード、同期、第三者サーバー経由はありません。
• 検索とショートカットの設定はローカル利用を前提にしています。新しいタブはあなたのものであるべきです。

【軽量で透明】
• 素の JavaScript、CSS、ブラウザ API で構築。フロントエンドフレームワーク、ビルド手順、大きなランタイム依存はありません。
• Chrome / Edge の新しいタブ拡張として動作し、Web ページとして直接開くこともできます。
• 完全にオープンソースで、日常利用にも、ブラウザ拡張、フロントエンド、AI 支援開発の学習にも向いています。

PlainTab は、こんな人に向いています：

• 情報フィードのノイズなしで、より美しい新しいタブを使いたい
• 日替わり壁紙、または自分の壁紙コレクションを管理したい
• 検索、ショートカット、ブックマークインポート、バックアップを、ホーム画面を散らかさずに使いたい
• ホーム画面が大量のショートカットタイルで埋まるのが嫌い
• 複雑なビルドツールのない、軽量で透明な拡張機能を使いたい
• 学習に使える、実在する完全なブラウザ拡張プロジェクトを探している

PlainTab がしないこと：

• ニュースフィード、トレンド一覧、推薦コンテンツは表示しません
• 広告、スポンサー枠、プロモーションカードは置きません
• アカウントシステム、ソーシャル機能、クラウドコンテンツフィードは追加しません
• ホーム画面を大量のショートカットで埋めません
• 自動再生動画やアニメーション背景は使いません

更新履歴

• v3.2.0 · 機能と性能の大型アップデート：コマンドパレット、RSS/API/ローカルフォルダ壁紙、WASM 壁紙テーマカラー、設定のインポート/エクスポート、検索履歴を追加；ストレージを v3.2 モデルへ移行し、権限と検索フォールバックを整理、起動・壁紙・ギャラリー・設定経路をモジュール化して最適化。

• v3.1.4 · ストレージ定数の命名をリファクタリングし、IDB 移行をクラッシュ耐性のある 2 フェーズ トランザクションに統合；新規ユーザーの移行スキップを最適化；バッチアップロードに重複排除と 12 枚上限を追加

• v3.1.3 · localStorage と IndexedDB のストレージバージョン移行を実装し、ストレージキー名を統一、ローカルモードでのサムネイル永続化ロジックを修正。

• v3.1.2 · Chrome Web Store 準拠のため _locales/pt を pt_BR に変更；changelog 形式を統一；ポルトガル語ファイルを pt_BR に改名。

• v3.1.0 · ローカル壁紙ギャラリー：複数壁紙のアップロード、削除、ローテーション、バッチ選択；パネルスタイル改善；ローカライズ強化；CSP 設定を厳格化。

• v3.0.5 · Bing API プロキシ更新、壁紙キャッシュ最適化、localStorage キー統一、16 言語 changelog 追加、ログとコード意味を改善。

• v3.0.4 · CSP の画像許可リストに Bing CDN ドメインが不足していたことによる壁紙読み込み失敗を修正。

• v3.0.3 · 初回描画パスを最適化、CSP セキュリティ強化、未使用コード整理、アイコンとプロモーション素材を追加。

• v3.0.2 · ダークモード適応背景、コード構造調整、多言語フッターを追加。

• v3.0.1 · Chrome Web Store の単一目的要件に対応するため Chrome Search API を統合。

• v3.0.0 · コアを書き直し：二層構造で新しいタブの白画面ちらつきを減らし、フォールバック背景を追加。

• v2.2.7 · 英語、日本語、韓国語、ロシア語、簡体字中国語 README を追加；デフォルト検索モードを常時表示に変更。

• v2.2.6 · 壁紙読み込みとサムネイルキャッシュを最適化、Bing API 定数を整理。

• v2.2.5 · 白画面を減らすため壁紙プリロードと初回フレーム最適化を追加。

• v2.2.4 · 壁紙ストレージを localStorage から IndexedDB に移行。

• v2.2.3 · ローカル壁紙アップロードを追加し、検索エンジン切り替え体験を改善。
