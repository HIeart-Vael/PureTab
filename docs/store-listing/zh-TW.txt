PlainTab 是一款讓你每次開啟新分頁都賞心悅目的瀏覽器擴充功能。

為什麼？因為你的新分頁值得一張好桌布，而不是廣告、新聞和捷徑連結。

為什麼選擇 PlainTab？

• 【桌布自由】
• 每日自動更新 Bing 美圖——全球雙端點並發競速，無論在哪都快。
• 極致的本地桌布載入速度——桌布存在你本機，開啟分頁瞬間直接讀取，比走網路快一個量級。
• 支援上傳最多 12 張本地桌布，每次開啟自動輪播。3×4 畫廊預覽，隨時刪、隨時換。

• 【絕對隱私保護】
• 無廣告、無追蹤、不收集任何資料。
• 所有桌布離線儲存在本機——不上傳、不同步、不經過任何伺服器。你的瀏覽行為只屬於你自己。

• 【極速，不打擾】
• 純原生 JavaScript，零外部依賴。無三方框架，瀏覽器開啟即呈現。
• 桌布在背景生成超輕量預覽快照，下次開啟新分頁時直接從本地瞬間讀取，不經過網路。

• 【靈活，不複雜】
• 16 種語言自動匹配。
• 搜尋欄：滑鼠懸停顯示 / 始終顯示 / 隱藏。
• 搜尋引擎：Google / Bing / 百度 / DuckDuckGo（Web 模式）。

• 【開源透明】
• 程式碼完全開源：https://github.com/kaininx/PlainTab
• 線上體驗：https://plaintab.kaininx.workers.dev

✨ 組合使用（開發者推薦）：
• 📌 新分頁：安裝本擴充功能，允許你最多設定 12 張不同桌布，不限大小。
• 📌 啟動頁：瀏覽器設為 https://plaintab.kaininx.workers.dev ，保持預設每日桌布模式。
• 📌 首頁按鈕：設為 https://kaininx.github.io/PlainTab/ ，這是本專案的 GitHub Pages，可額外設定你喜歡的桌布。

PlainTab 適合，如果你受夠了：
• 新分頁塞滿廣告、新聞和無用的捷徑連結
• 瀏覽器擴充功能在背景偷偷收集你的資料
• 擴充功能一多瀏覽器就卡，想要原生般的速度
• 每天開啟新分頁都是同一張圖，看膩了
