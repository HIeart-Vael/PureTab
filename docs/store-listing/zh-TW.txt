PlainTab 是一個好看、很快、很安靜的新分頁。

為什麼？因為你的新分頁應該先給你一張舒服的桌布，而不是廣告、新聞、熱門內容和一整屏捷徑。

PlainTab 做的事情很簡單：開啟新分頁，看到桌布，然後搜尋、輸入網址，或開啟自己的快捷連結。它不搶注意力，也不把瀏覽器首頁變成另一個資訊流。

為什麼選擇 PlainTab？

【開啟就有畫面】
• 極快的本機桌布載入速度——本機保存輕量預覽快照，每次開啟瞬間載入，減少白屏閃爍。
• 高清桌布會在背景平滑切換，既有畫面留在底層兜底，不需要等網路才看到背景。
• Bing 每日桌布支援快取和更新；只要使用過一次，後續開啟會更順。

【桌布自由】
• 支援 Bing 每日桌布，也支援上傳自己的圖片。
• 支援選擇本機桌布資料夾，適合已有桌布收藏的使用者。
• 支援 RSS 圖片來源和自訂 API，想簡單用可以簡單用，想折騰也有空間。
• 本機桌布最多支援 12 張輪播，提供 3×4 圖庫預覽，隨時刪除、隨時更換。

【安靜但不簡陋】
• 首頁保持克制：桌布、搜尋列、少量必要入口，不鋪滿元件。
• 搜尋列可調整位置、大小、圓角、透明度和顯示方式。
• 搜尋列支援始終顯示、需要時出現或隱藏，適合不同使用習慣。
• 快捷連結放在命令面板裡管理，支援搜尋、新增、編輯和書籤匯入，不佔用首頁畫面。
• 隱藏空間可以保存不想直接展示的連結，讓首頁繼續保持乾淨。

【隱私優先】
• 無廣告、無追蹤、不收集你的瀏覽資料。
• 本機上傳的桌布保存在瀏覽器本機儲存中，不上傳、不同步，也不經過第三方伺服器。
• 搜尋和快捷連結設定都以本機使用為主，你的新分頁應該屬於你自己。

【輕量透明】
• 使用原生 JavaScript、CSS 和瀏覽器 API 建構，沒有前端框架，沒有建置步驟，也沒有大型執行階段依賴。
• 可以作為 Chrome / Edge 擴充功能接管新分頁，也可以直接作為網頁開啟使用。
• 專案完全開源，適合日常使用，也適合學習瀏覽器擴充功能、前端專案和 AI 協作開發。

PlainTab 適合，如果你希望：

• 新分頁更好看，但不想被資訊流打擾
• 喜歡每日桌布，或者有自己的桌布收藏
• 想保留搜尋、快捷連結、書籤匯入和備份能力
• 不喜歡首頁被幾十個快捷方式鋪滿
• 想用一個輕量、透明、沒有複雜建置工具的擴充功能
• 想找一個真實完整的瀏覽器擴充功能專案來學習

PlainTab 不會做什麼：

• 不放新聞流、熱門榜或推薦內容
• 不放廣告、贊助卡片或推廣位
• 不加入帳號系統、社交功能或雲端內容流
• 不把幾十個快捷方式鋪滿首頁
• 不自動播放影片或動態背景
