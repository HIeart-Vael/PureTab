PlainTab es una extensión de navegador que hace que cada nueva pestaña sea un placer visual.

¿Por qué? Porque tu nueva pestaña merece un buen fondo de pantalla, no anuncios, noticias ni accesos directos.

¿Por qué elegir PlainTab?

• 【Libertad de fondos de pantalla】
• Imagen diaria de Bing actualizada automáticamente — dos endpoints globales en concurrencia competitiva, rápido sin importar dónde estés.
• Velocidad de carga local máxima — los fondos se almacenan en tu propio equipo, se leen al instante al abrir una pestaña, un orden de magnitud más rápido que cualquier carga por red.
• Sube hasta 12 fondos de pantalla locales, con rotación automática en cada nueva pestaña. Vista previa en galería 3×4, elimina o cambia cuando quieras.

• 【Privacidad absoluta】
• Sin anuncios, sin seguimiento, sin recopilación de datos.
• Todos los fondos se almacenan offline en tu equipo — no se suben, no se sincronizan, no pasan por ningún servidor. Tu actividad de navegación te pertenece solo a ti.

• 【Rápido, sin molestias】
• JavaScript puro nativo, cero dependencias externas. Sin frameworks de terceros, el navegador lo muestra al instante.
• Los fondos generan en segundo plano una vista previa ultraligera; la próxima vez que abras una nueva pestaña, se lee directamente desde el equipo local al instante, sin pasar por la red.

• 【Flexible, no complicado】
• Coincidencia automática en 16 idiomas.
• Barra de búsqueda: aparece al pasar el ratón / siempre visible / oculta.
• Motor de búsqueda: Google / Bing / Baidu / DuckDuckGo (modo Web).

• 【Transparencia y código abierto】
• Código completamente abierto: https://github.com/kaininx/PlainTab
• Pruébalo en línea: https://plaintab.kaininx.workers.dev

✨ Uso combinado (recomendado por el desarrollador):
• 📌 Nueva pestaña: instala esta extensión, te permite configurar hasta 12 fondos de pantalla diferentes, sin límite de tamaño.
• 📌 Página de inicio: configura el navegador en https://plaintab.kaininx.workers.dev para mantener el modo diario predeterminado.
• 📌 Botón de inicio: configúralo en https://kaininx.github.io/PlainTab/ , que es la GitHub Pages de este proyecto, para establecer fondos adicionales a tu gusto.

PlainTab es para ti, si estás harto de:
• Nuevas pestañas llenas de anuncios, noticias y accesos directos inútiles
• Extensiones que recopilan tus datos a escondidas en segundo plano
• Navegadores que se vuelven lentos con muchas extensiones, cuando lo que quieres es velocidad nativa
• Ver la misma imagen cada vez que abres una nueva pestaña

Registro de cambios

• v3.1.3 · Implementada la migración de versión de almacenamiento para localStorage e IndexedDB, unificada la nomenclatura de claves, corregida la lógica de persistencia de miniaturas en modo local.

• v3.1.2 · Se renombró _locales/pt a pt_BR para cumplir con Chrome Web Store; se unificó el formato del changelog al estilo compacto; se renombraron los archivos portugueses a pt_BR para mantener consistencia.

• v3.1.1 · Se cambió la API de Bing al modo de concurrencia Promise.any con un tiempo de espera AbortController de 8 segundos; se actualizaron los 16 README de idiomas para reflejar la nueva estrategia de obtención.

• v3.1.0 · Galería local de fondos: sube, elimina y rota múltiples fondos locales, selección por lotes; refactorización del código; mejora de estilos en paneles; localización multilingüe mejorada; corrección de CSP eliminando dominios de API JSON.

• v3.0.5 · Actualización del proxy de la API de Bing (peticiones regionalizadas + deduplicación de imágenes); optimización del caché de fondos; unificación de nombres de claves en localStorage; nuevo changelog en 16 idiomas; mejora de logs y semántica del código.

• v3.0.4 · Corrección de fallo al cargar fondos: la lista blanca CSP img-src no incluía el dominio CDN de Bing, por lo que las imágenes redirigidas por el proxy API eran bloqueadas por el navegador.

• v3.0.3 · Omitir CSSOM para acelerar el renderizado del primer fotograma, refuerzo de seguridad CSP, limpieza de código muerto, nuevos iconos SVG y material promocional.

• v3.0.2 · Fondo adaptativo en modo oscuro, reestructuración del código, barra de información multilingüe en la parte inferior.

• v3.0.1 · Integración de Chrome Search API para superar la auditoría de cumplimiento de propósito único de Chrome Web Store.

• v3.0.0 · Reescritura completa: arquitectura de doble capa para eliminar el parpadeo de pantalla blanca en nuevas pestañas, añadido fondo gris de respaldo.

• v2.2.7 · Nuevos README en inglés, japonés, coreano, ruso y chino simplificado; modo de búsqueda predeterminado cambiado a "siempre visible".

• v2.2.6 · Optimización de la lógica de carga de fondos y del caché de miniaturas, reorganización de constantes de la API de Bing.

• v2.2.5 · Precarga de fondos y optimización del primer fotograma, eliminando la pantalla blanca en nuevas pestañas.

• v2.2.4 · Migración del almacenamiento de fondos de localStorage a IndexedDB.

• v2.2.3 · Subida de fondos locales, mejora en la experiencia de cambio de motor de búsqueda.

• v1.2.0 · Primera publicación: sistema de fondos, interfaz en 16 idiomas, localización Chrome i18n, licencia MIT.
