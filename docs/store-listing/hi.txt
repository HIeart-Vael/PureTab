PlainTab एक सुंदर, तेज़ और शांत नया टैब पेज है।

क्यों? क्योंकि आपका नया टैब पहले एक आरामदायक वॉलपेपर दिखाए, न कि विज्ञापन, समाचार, ट्रेंड या शॉर्टकट से भरी स्क्रीन।

PlainTab एक सरल काम करता है: नया टैब खोलें, वॉलपेपर देखें, फिर खोजें, URL टाइप करें या अपने शॉर्टकट खोलें। यह आपका ध्यान नहीं खींचता और ब्राउज़र होमपेज को एक और कंटेंट फीड नहीं बनाता।

PlainTab क्यों चुनें?

【खुलते ही चित्र】
• बेहद तेज़ लोकल वॉलपेपर लोडिंग: PlainTab हल्का प्रीव्यू स्नैपशॉट लोकल रूप से सहेजता है और हर नए टैब पर तुरंत लोड करता है, जिससे सफेद स्क्रीन झिलमिलाहट कम होती है।
• हाई-रेज़ोल्यूशन वॉलपेपर बैकग्राउंड में स्मूथ तरीके से बदलते हैं, जबकि मौजूदा चित्र नीचे fallback के रूप में रहता है। इसलिए बैकग्राउंड देखने के लिए नेटवर्क का इंतज़ार नहीं करना पड़ता।
• Bing दैनिक वॉलपेपर cache और update को सपोर्ट करते हैं; एक बार उपयोग करने के बाद अगली बार खुलना और भी सहज लगता है।

【वॉलपेपर की आज़ादी】
• Bing दैनिक वॉलपेपर और आपकी अपनी अपलोड की गई तस्वीरों को सपोर्ट करता है।
• लोकल वॉलपेपर फ़ोल्डर चुनने की सुविधा, अगर आपके पास पहले से वॉलपेपर संग्रह है।
• RSS image sources और custom APIs सपोर्ट करता है: सरल उपयोग के लिए सरल, और प्रयोग करने के लिए लचीला।
• लोकल वॉलपेपर में अधिकतम 12 चित्रों की rotation और 3×4 gallery preview सपोर्ट है, जिन्हें कभी भी हटाया या बदला जा सकता है।

【शांत, लेकिन खाली नहीं】
• होमपेज संयमित रहता है: वॉलपेपर, search bar और कुछ ज़रूरी controls, बिना widgets से पेज भरने के।
• search bar की position, size, corner radius, opacity और visibility mode बदली जा सकती है।
• search bar हमेशा दिखाई दे सकता है, ज़रूरत पर दिख सकता है या छिपा रह सकता है।
• शॉर्टकट command palette में manage होते हैं, search, add, edit और bookmark import के साथ, इसलिए वे वॉलपेपर को ढकते नहीं।
• hidden space उन links को रखने देता है जिन्हें आप सीधे नहीं दिखाना चाहते, ताकि homepage साफ़ रहे।

【Privacy पहले】
• कोई विज्ञापन नहीं, कोई tracking नहीं, browsing data collection नहीं।
• लोकल रूप से upload किए गए वॉलपेपर browser के local storage में रहते हैं। वे upload, sync या third-party servers से होकर नहीं जाते।
• search और shortcut settings local use के लिए डिज़ाइन की गई हैं। आपका नया टैब आपका होना चाहिए।

【हल्का और पारदर्शी】
• Vanilla JavaScript, CSS और browser APIs से बना है, बिना frontend framework, बिना build step और बिना बड़े runtime dependency के।
• Chrome / Edge new tab extension के रूप में काम करता है, और सीधे web page के रूप में भी खोला जा सकता है।
• पूरी तरह open source, रोज़मर्रा के उपयोग और browser extension, frontend project तथा AI-assisted engineering सीखने के लिए उपयुक्त।

PlainTab आपके लिए है, यदि आप चाहते हैं:

• information feed के शोर के बिना सुंदर नया टैब
• दैनिक वॉलपेपर या अपनी wallpaper collection manage करने की जगह
• search, shortcuts, bookmark import और backup बिना homepage clutter के
• ऐसा homepage जो दर्जनों shortcut tiles से ढका न हो
• complex build tools के बिना हल्का और transparent extension
• सीखने के लिए वास्तविक और पूर्ण browser extension project

PlainTab क्या नहीं करेगा:

• news feeds, trending lists या recommended content नहीं
• ads, sponsored cards या promotional slots नहीं
• account system, social features या cloud content stream नहीं
• homepage पर shortcut tiles की दीवार नहीं
• autoplay video या animated backgrounds नहीं
