PlainTab은 아름답고 빠르며 조용한 새 탭 페이지입니다.

왜냐하면 새 탭은 광고, 뉴스, 인기 목록, 화면 가득한 바로가기보다 먼저 편안한 배경화면을 보여줘야 하기 때문입니다.

PlainTab은 한 가지 단순한 일을 합니다. 새 탭을 열면 배경화면이 보이고, 그다음 검색하거나 URL을 입력하거나 자신의 바로가기를 열 수 있습니다. 주의를 빼앗지 않고 브라우저 홈을 또 하나의 콘텐츠 피드로 만들지 않습니다.

왜 PlainTab인가요?

【열자마자 보이는 화면】
• 매우 빠른 로컬 배경화면 로딩: PlainTab은 가벼운 미리보기 스냅샷을 로컬에 저장하고 새 탭을 열 때마다 즉시 불러와 흰 화면 깜박임을 줄입니다.
• 고해상도 배경화면은 백그라운드에서 부드럽게 전환되고, 기존 이미지는 아래 레이어에 fallback으로 남아 네트워크를 기다리지 않아도 배경을 볼 수 있습니다.
• Bing 일일 배경화면은 캐시와 업데이트를 지원합니다. 한 번 사용한 뒤에는 이후 열림이 더 부드럽게 느껴집니다.

【배경화면 자유】
• Bing 일일 배경화면과 사용자가 업로드한 이미지를 지원합니다.
• 로컬 배경화면 폴더 선택을 지원해 이미 배경화면 컬렉션이 있는 사용자에게 적합합니다.
• RSS 이미지 소스와 사용자 지정 API를 지원합니다. 간단히 쓰고 싶을 때는 간단하게, 더 꾸미고 싶을 때는 유연하게 사용할 수 있습니다.
• 로컬 배경화면은 최대 12장 회전을 지원하며 3×4 갤러리 미리보기에서 언제든 삭제하거나 교체할 수 있습니다.

【조용하지만 비어 있지 않음】
• 홈 화면은 절제되어 있습니다. 배경화면, 검색창, 필요한 몇 가지 진입점만 있고 위젯으로 가득 차지 않습니다.
• 검색창은 위치, 크기, 모서리 반경, 투명도, 표시 방식을 조정할 수 있습니다.
• 항상 표시, 필요할 때만 표시, 숨김 중에서 선택할 수 있습니다.
• 바로가기는 명령 패널에서 관리되며 검색, 추가, 편집, 북마크 가져오기를 지원해 배경화면을 가리지 않습니다.
• 숨김 공간에는 직접 보여주고 싶지 않은 링크를 저장해 홈 화면을 깔끔하게 유지할 수 있습니다.

【개인정보 우선】
• 광고 없음, 추적 없음, 브라우징 데이터 수집 없음.
• 로컬로 업로드한 배경화면은 브라우저 로컬 저장소에 보관됩니다. 업로드되거나 동기화되거나 타사 서버를 거치지 않습니다.
• 검색과 바로가기 설정은 로컬 사용을 중심으로 설계되었습니다. 새 탭은 사용자 자신의 것이어야 합니다.

【가볍고 투명함】
• 순수 JavaScript, CSS, 브라우저 API로 만들어졌으며 프론트엔드 프레임워크, 빌드 단계, 대형 런타임 의존성이 없습니다.
• Chrome / Edge 새 탭 확장 프로그램으로 작동하며 웹 페이지로도 직접 열 수 있습니다.
• 완전한 오픈소스로, 일상 사용은 물론 브라우저 확장, 프론트엔드 프로젝트, AI 협업 개발 학습에도 적합합니다.

PlainTab은 이런 분께 적합합니다:

• 정보 피드의 소음 없이 더 보기 좋은 새 탭을 원할 때
• 일일 배경화면을 쓰거나 자신만의 배경화면 컬렉션을 관리하고 싶을 때
• 검색, 바로가기, 북마크 가져오기, 백업 기능을 홈 화면을 어지럽히지 않고 쓰고 싶을 때
• 홈 화면이 수십 개의 바로가기 타일로 덮이는 것이 싫을 때
• 복잡한 빌드 도구가 없는 가볍고 투명한 확장 프로그램을 원할 때
• 학습할 수 있는 실제적이고 완성도 있는 브라우저 확장 프로젝트를 찾을 때

PlainTab이 하지 않는 것:

• 뉴스 피드, 인기 목록, 추천 콘텐츠 없음
• 광고, 스폰서 카드, 프로모션 영역 없음
• 계정 시스템, 소셜 기능, 클라우드 콘텐츠 스트림 없음
• 홈 화면을 바로가기 타일로 가득 채우지 않음
• 자동 재생 동영상 또는 애니메이션 배경 없음

변경 내역

• v3.2.0 · 대규모 기능 및 성능 업데이트: 명령 패널, RSS/API/로컬 폴더 배경화면, WASM 배경화면 테마 색상, 설정 가져오기/내보내기와 검색 기록 추가; 스토리지를 v3.2 모델로 마이그레이션하고 권한과 검색 fallback을 정리했으며 시작, 배경화면, 갤러리, 설정 경로를 모듈화하고 최적화.

• v3.1.4 · 스토리지 상수 이름을 리팩터링하고 IDB 마이그레이션을 충돌 복구 가능한 2단계 트랜잭션으로 통합; 신규 사용자 마이그레이션 건너뛰기 최적화; 일괄 업로드 중복 제거 및 12장 제한 추가

• v3.1.3 · localStorage와 IndexedDB를 위한 스토리지 버전 마이그레이션 구현, 스토리지 키 이름 통일, 로컬 모드에서 썸네일 지속성 로직 수정.

• v3.1.2 · Chrome Web Store 준수를 위해 _locales/pt를 pt_BR로 변경; changelog 형식 통일; 포르투갈어 파일을 pt_BR로 변경.

• v3.1.1 · Bing API를 Promise.any 경쟁 모드와 8초 AbortController timeout으로 전환; 16개 언어 README 업데이트.

• v3.1.0 · 로컬 배경화면 갤러리: 여러 배경화면 업로드, 삭제, 회전, 일괄 선택; 패널 스타일 개선; 로컬라이제이션 강화; CSP 설정 강화.

• v3.0.5 · Bing API 프록시 업그레이드, 배경화면 캐시 최적화, localStorage 키 통일, 16개 언어 changelog 추가, 로그와 코드 의미 개선.

• v3.0.4 · CSP 이미지 허용 목록에 Bing CDN 도메인이 없어 발생한 배경화면 로딩 실패 수정.

• v3.0.3 · 첫 프레임 렌더링 경로 최적화, CSP 보안 강화, 사용하지 않는 코드 정리, 아이콘과 홍보 자료 추가.

• v3.0.2 · 다크 모드 적응형 배경, 코드 구조 조정, 다국어 하단 안내 추가.

• v3.0.1 · Chrome Web Store 단일 목적 요구사항을 충족하기 위해 Chrome Search API 통합.

• v3.0.0 · 핵심 재작성: 이중 레이어 구조로 새 탭 흰 화면 깜박임 감소, fallback 배경 추가.

• v2.2.7 · 영어, 일본어, 한국어, 러시아어, 중국어 간체 README 추가; 기본 검색 모드를 항상 표시로 변경.

• v2.2.6 · 배경화면 로딩과 썸네일 캐시 최적화, Bing API 상수 정리.

• v2.2.5 · 흰 화면을 줄이기 위해 배경화면 사전 로딩과 첫 프레임 최적화 추가.

• v2.2.4 · 배경화면 저장소를 localStorage에서 IndexedDB로 마이그레이션.

• v2.2.3 · 로컬 배경화면 업로드 추가 및 검색 엔진 전환 경험 개선.

• v1.2.0 · 첫 릴리스: 배경화면 시스템, 16개 언어 UI, Chrome i18n 로컬라이제이션, MIT 라이선스.
