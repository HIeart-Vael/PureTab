PlainTab은 새 탭을 열 때마다 눈이 즐거운 브라우저 확장 프로그램입니다.

왜? 새 탭 페이지는 광고, 뉴스, 바로가기 링크가 아니라 예쁜 배경화면을 받을 자격이 있기 때문입니다.

PlainTab을 선택해야 하는 이유:

• 【배경화면의 자유】
• Bing 일일 배경화면 자동 업데이트 — 전역 이중 엔드포인트가 Promise.any로 경쟁하여 어디에 있든 빠르게 제공됩니다.
• 극강의 로컬 배경화면 로딩 속도 — 배경화면이 사용자 컴퓨터에 저장되어 새 탭을 열면 네트워크보다 훨씬 빠르게 즉시 로딩됩니다.
• 최대 12장의 로컬 배경화면 업로드 지원, 새 탭을 열 때마다 자동 순환. 3×4 갤러리 미리보기로 언제든 삭제 및 교체 가능.

• 【완벽한 개인정보 보호】
• 광고 없음, 추적 없음, 어떠한 데이터도 수집하지 않습니다.
• 모든 배경화면은 사용자 기기에 오프라인 저장 — 업로드, 동기화, 서버 경유가 전혀 없습니다. 사용자의 브라우징 행동은 오직 사용자만의 것입니다.

• 【초고속, 방해 없음】
• 순수 네이티브 JavaScript, 외부 의존성 제로. 서드파티 프레임워크 불필요. 브라우저를 열면 즉시 표시됩니다.
• 배경화면은 백그라운드에서 초경량 미리보기 스냅샷을 생성합니다. 다음에 새 탭을 열면 로컬에서 즉시 로딩되며 네트워크를 거치지 않습니다.

• 【유연하면서도 복잡하지 않음】
• 16개 언어 자동 매칭.
• 검색창: 마우스 호버 시 표시 / 항상 표시 / 숨기기.
• 검색 엔진: Google / Bing / Baidu / DuckDuckGo (웹 모드).

• 【오픈소스 & 투명성】
• 코드 완전 공개: https://github.com/kaininx/PlainTab
• 온라인 체험: https://plaintab.netlify.app

✨ 조합하여 사용하기 (개발자 추천):
• 📌 새 탭: 본 확장 프로그램을 설치하여 최대 12장의 다양한 배경화면을 설정하세요. 크기 제한 없음.
• 📌 시작 페이지: 브라우저 홈페이지를 https://plaintab.netlify.app 로 설정하고 기본 일일 Bing 배경화면 모드를 유지하세요.
• 📌 홈 버튼: https://kaininx.github.io/PlainTab/ 로 설정하세요. 이 프로젝트의 GitHub Pages로, 원하는 배경화면을 추가로 설정할 수 있습니다.

PlainTab이 적합한 분들:
• 새 탭에 광고, 뉴스, 쓸데없는 바로가기가 가득한 것에 지친 분
• 브라우저 확장 프로그램이 백그라운드에서 몰래 데이터를 수집하는 것이 걱정되는 분
• 확장 프로그램이 많으면 브라우저가 느려져서 네이티브 속도를 원하는 분
• 매일 같은 배경화면만 보고 질린 분

• v3.1.0 · 로컬 배경화면 갤러리: 여러 배경화면 업로드/삭제/순환, 일괄 파일 선택. 코드 리팩토링, 패널 스타일 개선, 다국어 현지화 강화. JSON API 도메인을 CSP에서 제거하여 수정.

• v3.0.5 · Bing API 프록시 업그레이드 (지역화 요청 + 이미지 중복 제거). 배경화면 캐시 최적화, localStorage 키 이름 통일. 16개 언어 체인지로그 추가. 로깅 및 코드 의미론 개선.

• v3.0.4 · 배경화면 로딩 실패 수정: CSP img-src 허용 목록에 Bing CDN 도메인이 누락되어 프록시 API 리디렉션 후 이미지가 브라우저에서 차단되는 문제 해결.
