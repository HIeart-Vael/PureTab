PlainTab 是一款让你每次打开新标签页都赏心悦目的浏览器扩展。

为什么？因为你的新标签页值得一张好壁纸，而不是广告、新闻和快捷链接。

为什么选择 PlainTab？

• 【壁纸自由】
• 每日自动更新必应美图——全球双端点并发竞速，无论在哪都快。
• 极致的本地壁纸加载速度——壁纸存你本机，打开标签页瞬间直接读取，比走网络快一个量级。
• 支持上传最多 12 张本地壁纸，每次打开自动轮播。3×4 画廊预览，随时删、随时换。

• 【绝对隐私保护】
• 无广告、无追踪、不收集任何数据。
• 所有壁纸离线存储在本机——不上传、不同步、不经过任何服务器。你的浏览行为只属于你自己。

• 【极速，不打扰】
• 纯原生 JavaScript，零外部依赖。无三方框架，浏览器打开即呈现。
• 壁纸在后台生成超轻量预览快照，下次打开新标签页时直接从本地瞬间读取，不经过网络。

• 【灵活，不复杂】
• 16 种语言自动匹配。
• 搜索栏：鼠标悬停出现 / 始终显示 / 隐藏。
• 搜索引擎：Google / Bing / 百度 / DuckDuckGo（Web 模式）。

• 【开源透明】
• 代码完全开源：https://github.com/kaininx/PlainTab
• 在线体验：https://plaintab.netlify.app

✨ 组合使用（开发者推荐）：
• 📌 新标签页：安装本扩展，允许你最多设置 12 张不同壁纸，不限大小。
• 📌 启动页：浏览器设为 https://plaintab.netlify.app ，保持默认每日壁纸模式。
• 📌 主页按钮：设为 https://kaininx.github.io/PlainTab/ ，这是本项目的 GitHub Pages，可额外设置你喜欢的壁纸。

PlainTab 适合，如果你受够了：
• 新标签页塞满广告、新闻和无用的快捷链接
• 浏览器扩展在后台偷偷收集你的数据
• 扩展一多浏览器就卡，想要原生般的速度
• 每天打开新标签页都是同一张图，看腻了
