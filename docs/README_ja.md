## これは何か

PlainTab はブラウザの新しいタブです。

やることはシンプルです。新しいタブを開いたとき、まずきれいな壁紙が目に入り、そのまま検索したり、URL を入力したり、自分のショートカットリンクを開いたりできます。

ニュースフィードも、画面いっぱいのウィジェットも、天気も、あいさつも、ToDo リストもありません。あなたの注意をどこかへ引きずっていくこともありません。むしろ、きれいなデスクトップのようなものです。必要なときはそこにあり、必要でないときは邪魔をしません。

## 試してみる価値がある理由

### 開いた瞬間に絵がある

PlainTab で一番感じるのは速さです。

多くの新しいタブは、開いたときにまず空白が一瞬ちらつき、少し待ってから画像が現れます。PlainTab はその感覚をできるだけ排除します。一度使ってブラウザにキャッシュがあれば、次に開いたとき、壁紙はまるでずっとそこにあったかのように素早く表示されます。

これは「パラメータ上のパフォーマンス最適化」ではなく、実際に感じ取れる滑らかさです。

### 第一印象が心地いい

デフォルトのインターフェースはとても控えめです。壁紙、検索バー、いくつかの小さなボタンだけ。

開いた瞬間に情報を詰め込まれることも、ブラウザのホームを別の作業台に変えられることもありません。その美しさは「少なさ」から来ていて、「多さ」からではありません。

### 壁紙の遊び方がたくさん

Bing の今日の壁紙をそのまま使ってもいいし、自分の画像をアップロードしてもいいし、PC 内の壁紙フォルダを選んでもいい。

いろいろ触るのが好きなら、RSS 画像ソースをつないだり、自分の画像 API をつないだりもできます。シンプルに使うのも簡単だし、深く遊ぶ余地もあります。

### 検索がとてもスムーズ

検索バーは好きな場所に置けて、サイズ、角丸、透明度、表示方法も調整できます。

常に表示してもいいし、普段は隠して必要なときだけ出すこともできます。見た目は軽やかですが、日常的にとても便利です。

### ショートカットリンクがホームを占領しない

多くの新しいタブはショートカットを画面いっぱいに並べます。PlainTab はそうしません。

ショートカットリンクはコマンドパネルの中に収めています。検索、追加、編集、ブックマークのインポートができ、一部のリンクを非表示スペースに入れることもできます。ホームはきれいなままで、機能は何も減っていません。

### 設定は十分、でもうるさくない

PlainTab では壁紙、検索バー、ショートカットキー、バックアップ、言語、インターフェースの細部を調整できます。

でも、その設定がホームに積み重なることはありません。普段目にするのは静かな新しいタブ。調整したくなったときに、設定パネルを開けばいいのです。

## これは AI 協働プロジェクトです

PlainTab は学びにもとても適しています。

デモ止まりのおもちゃプロジェクトではなく、実際のインターフェース、ユーザー設定、データ保存、インポート/エクスポート、多言語、ブラウザ拡張環境、そして通常の Web 環境を備えた、本格的な新しいタブ拡張機能です。

このプロジェクトは、開発・整理・ドキュメント作成・イテレーションの過程で AI 協働を多用して完成しました。優れた学習サンプルとして活用できます：

- AI が実際のプロジェクト開発にどう関わるかを見てみたい。
- ブラウザ拡張機能がどう作られているか学びたい。
- フロントエンドに入門したいけど、いきなり複雑なフレームワークに直面したくない。
- アイデア・機能・体験・ドキュメントまで、かなり完成度の高い小さなプロジェクトを研究したい。

コードは複雑なビルドツールに依存せず、ファイルを開けば読めて、ページを開けば動きます。初心者にとっては、最初からたくさんの依存をインストールするよりずっと親切です。

## どんな人に向いているか

PlainTab はこんな人に向いています：

- もっと美しい新しいタブに替えたい。
- ブラウザを開いてすぐ情報の洪水に遭いたくない。
- 今日の壁紙が好き、または自分だけの壁紙コレクションを持っている。
- 軽量でいて機能が整ったスタートページがほしい。
- AI 協働プロジェクトのリアルで完全な学習サンプルを探している。
- ブラウザ拡張機能やフロントエンド開発に入門したい。

## やらないこと

PlainTab はこれからも控えめであり続けます。多くの人が慣れ親しんでいる機能でも、このプロジェクトには必ずしも合いません。

おそらくサポートしないもの：

- ニュースフィード、トレンドランキング、おすすめコンテンツ。
- 起動時広告、プロモーションカード、スポンサー枠。
- 大きな天気・カレンダー・ToDo パネル。
- 動画の自動再生や動的な背景。
- アカウントシステム、ソーシャル機能、クラウドコンテンツストリーム。
- ホームを数十のショートカットで埋め尽くすこと。
- Safari 版。追加のリリースとメンテナンスコストが必要で、今の個人プロジェクトにはまだ現実的ではありません。

これらの機能が良くないのではなく、それらが PlainTab を別の何かに変えてしまうからです。

新しいタブを開くのが、ただ少し静かになりたい、何か検索したい、壁紙を一目見たい——それだけなら、PlainTab はきっとあなたの味方です。

## これからやるかもしれないこと

PlainTab をこれからこんな方向に育てていきたいと思っています：

- より安定した壁紙ソース。
- より快適な設定体験。
- よりわかりやすい学習ドキュメント。
- 初心者が読みやすいコードの説明。
- より完全な AI 協働開発の記録。
- Firefox 対応の可能性。

Firefox 対応は今すぐお約束はできません。ブラウザ拡張の環境・権限・API の挙動が Chrome / Edge とは異なる可能性があるからです。でも、将来的に条件が整えば、PlainTab を Chromium 系ブラウザだけに留めたくはありません。

もし Firefox 拡張やブラウザ互換性、AI 支援開発にちょうど興味があれば、ぜひ一緒に研究しましょう。このプロジェクトは使いながら学ぶのにぴったりです。

## 機能一覧

| 機能 | 説明 |
|------|------|
| 新しいタブの置き換え | インストール後、ブラウザの新しいタブを引き継ぎます |
| オンラインスタートページ | そのまま Web ページとしても使えます |
| 今日の壁紙 | Bing の今日の壁紙に対応 |
| ローカル壁紙 | 画像のアップロードとローカルフォルダの選択に対応 |
| RSS / API 壁紙 | さらに多くの画像ソースを接続できます |
| 検索バー | 位置・サイズ・スタイル・表示方法を調整可能 |
| 検索履歴 | 最近の検索を保存、またはオフにもできます |
| ショートカットリンク | コマンドパネルで管理、ホームを占有しません |
| 非表示スペース | 表示したくないリンクを別に保存 |
| 設定パネル | インターフェース・壁紙・ショートカット・データを調整 |
| データバックアップ | エクスポート・インポート・暗号化バックアップに対応 |
| 多言語 | 多言語インターフェースに対応 |
| AI 協働 | AI がプロジェクト開発に関わる様子を学べます |

## クイックスタート

### 拡張機能のインストール

[Chrome Web Store で PlainTab をインストール](https://chromewebstore.google.com/detail/plaintab-%C2%B7-minimal-new-ta/jhpfjcefcmooplmaimgdafohdlhacjdo)

### オンライン体験

[plaintab.kaininx.workers.dev](https://plaintab.kaininx.workers.dev)

### ローカルで実行

```bash
git clone https://github.com/kaininx/PlainTab.git
```

拡張モード：

1. `chrome://extensions` を開く
2. デベロッパーモードを有効にする
3. 「パッケージ化されていない拡張機能を読み込む」を選択
4. PlainTab のプロジェクトディレクトリを選択

Web モード：

ブラウザで直接 `index.html` を開きます。

依存のインストールもビルドも不要です。

## プロジェクト構成

```text
PlainTab/
├── index.html              # 新しいタブと Web エントリポイント
├── manifest.json           # ブラウザ拡張機能の設定
├── css/                    # ページスタイル
├── js/                     # ページロジック
├── js/wallpaper/           # 壁紙関連機能
├── _locales/               # 拡張機能の多言語
├── docs/                   # ドキュメント
├── icon/                   # アイコン
└── imgs/                   # スクリーンショットとストア素材
```
