## Что это

PlainTab — это страница новой вкладки для браузера.

Она делает простую вещь: когда вы открываете новую вкладку, вы сначала видите красивые обои, а затем можете сразу искать, вводить адрес или открывать свои быстрые ссылки.

Здесь нет новостной ленты, нет заполняющих экран виджетов, нет погоды, приветствий, списка дел и ничего, что отвлекало бы ваше внимание. Это скорее как чистый рабочий стол: он здесь, когда нужен, и не мешает, когда не нужен.

## Почему стоит попробовать

### Изображение сразу при открытии

Самое заметное в PlainTab — это скорость.

Многие страницы новой вкладки при открытии сначала показывают пустой экран, а потом появляется картинка. PlainTab старается избежать этого ощущения. Как только вы воспользовались им один раз и в браузере уже есть кеш, при следующем открытии обои появляются очень быстро — как будто они всегда были там.

Это не та «оптимизация производительности», которая существует только в спецификациях — это плавность, которую вы чувствуете напрямую.

### Приятно с первого взгляда

Интерфейс по умолчанию очень сдержанный: обои, строка поиска, несколько маленьких кнопок.

Он не заваливает вас информацией при открытии и не превращает домашнюю страницу браузера в ещё одно рабочее место. Его красота исходит от минимализма, а не от нагромождения.

### Много возможностей для обоев

Вы можете использовать ежедневные обои Bing, загружать свои изображения или выбрать папку с обоями на своём компьютере.

Если вам нравится экспериментировать, можно подключить RSS-источники изображений или свой собственный API изображений. Можно пользоваться просто, а можно глубоко настроить.

### Удобный поиск

Строку поиска можно разместить там, где вам нравится, а также настроить размер, скругление, прозрачность и режим отображения.

Вы можете оставить её всегда видимой или скрывать, когда она не нужна. Она выглядит лёгкой, но очень удобна в повседневном использовании.

### Быстрые ссылки не загромождают страницу

Многие страницы новой вкладки заполняют экран быстрыми ссылками. PlainTab так не делает.

Он помещает быстрые ссылки в командную палитру. Вы можете искать, добавлять, редактировать, импортировать закладки, а также помещать некоторые ссылки в скрытое пространство. Главная страница остаётся чистой, но функциональность не теряется.

### Много настроек, но не навязчиво

В PlainTab можно настроить обои, строку поиска, горячие клавиши, резервное копирование, язык, детали интерфейса.

Но эти настройки не загромождают главную страницу. Обычно вы видите спокойную страницу новой вкладки; когда нужно что-то настроить — открываете панель настроек.

## Это проект с участием ИИ

PlainTab также отлично подходит для обучения.

Это не игрушечный проект, оставшийся на уровне демонстрации, а полноценное расширение для новой вкладки: с настоящим интерфейсом, пользовательскими настройками, сохранением данных, импортом и экспортом, мультиязычностью, поддержкой как среды браузерного расширения, так и обычной веб-страницы.

Этот проект был в значительной степени создан с использованием ИИ-ассистентов в процессе разработки, организации, документирования и итераций. Вы можете рассматривать его как отличный учебный образец:

- Чтобы увидеть, как ИИ участвует в реальной разработке проекта.
- Чтобы узнать, как делается браузерное расширение.
- Чтобы начать заниматься фронтендом, не сталкиваясь сразу со сложными фреймворками.
- Чтобы изучить небольшой проект, достаточно полный по циклу: от идеи и функционала до UX и документации.

Его код не зависит от сложных инструментов сборки — открываете файл и читаете, открываете страницу и она работает. Для новичков это гораздо дружелюбнее, чем сначала устанавливать кучу зависимостей.

## Кому подойдёт

PlainTab подходит тем:

- Кто хочет сменить страницу новой вкладки на более красивую.
- Кому не нравится видеть кучу информации при открытии браузера.
- Кто любит ежедневные обои или имеет свою коллекцию обоев.
- Кто хочет лёгкую, но функционально полную стартовую страницу.
- Кто ищет реальный, завершённый проект с ИИ-коллаборацией для изучения.
- Кто хочет начать заниматься браузерными расширениями или фронтенд-проектами.

## Что он не будет делать

PlainTab продолжит оставаться сдержанным. Некоторые функции, даже если многие с ними знакомы, не обязательно подходят для этого проекта.

Скорее всего, не будет поддерживаться:

- Новостные ленты, рейтинги, рекомендательный контент.
- Реклама при открытии, рекламные карточки, спонсорские места.
- Крупные панели погоды, календаря, задач.
- Автоматически воспроизводимые видео или анимированные фоны.
- Системы учётных записей, социальные функции, облачные потоки контента.
- Заполнение главной страницы десятками быстрых ссылок.
- Версия для Safari. Она потребовала бы дополнительных затрат на публикацию и поддержку, что пока нереалистично для личного проекта.

Не то чтобы эти функции плохи — просто они превратили бы PlainTab во что-то другое.

Если вы открываете новую вкладку, чтобы просто немного успокоиться, что-то поискать или взглянуть на обои, то PlainTab должен быть на вашей стороне.

## Что возможно в будущем

Я бы хотел, чтобы PlainTab продолжал расти в этих направлениях:

- Более стабильные источники обоев.
- Более приятный опыт настроек.
- Более понятная учебная документация.
- Более доступные для новичков пояснения кода.
- Более полная запись процесса ИИ-ассистированной разработки.
- Возможная поддержка Firefox.

Поддержку Firefox пока нельзя прямо обещать, потому что среда расширений, разрешения и поведение API могут отличаться от Chrome / Edge. Но если в будущем условия будут подходящими, я бы хотел, чтобы PlainTab не ограничивался браузерами на Chromium.

Если вас как раз интересуют расширения для Firefox, кроссбраузерная совместимость или ИИ-ассистированная разработка — добро пожаловать к совместному исследованию. Этот проект хорошо подходит для того, чтобы учиться на практике.

## Обзор функций

| Функция | Описание |
|------|------|
| Замена новой вкладки | После установки перехватывает новую вкладку браузера |
| Онлайн-стартовая | Можно также использовать напрямую как веб-страницу |
| Ежедневные обои | Поддерживает ежедневные обои Bing |
| Локальные обои | Поддерживает загрузку изображений и выбор локальной папки |
| RSS / API обои | Можно подключать дополнительные источники изображений |
| Строка поиска | Настраиваемые положение, размер, стиль и режим отображения |
| История поиска | Можно сохранять недавние поиски или отключить |
| Быстрые ссылки | Управление через командную палитру, не занимая главную страницу |
| Скрытое пространство | Отдельное хранение ссылок, которые не должны отображаться |
| Панель настроек | Настройка интерфейса, обоев, горячих клавиш и данных |
| Резервное копирование | Поддержка экспорта, импорта и шифрованного резервного копирования |
| Мультиязычность | Поддержка многоязычного интерфейса |
| ИИ-коллаборация | Подходит для изучения участия ИИ в разработке проектов |

## Быстрый старт

### Установка расширения

[Установить PlainTab из Chrome Web Store](https://chromewebstore.google.com/detail/plaintab-%C2%B7-minimal-new-ta/jhpfjcefcmooplmaimgdafohdlhacjdo)

### Онлайн-демо

[plaintab.kaininx.workers.dev](https://plaintab.kaininx.workers.dev)

### Локальный запуск

```bash
git clone https://github.com/kaininx/PlainTab.git
```

Режим расширения:

1. Откройте `chrome://extensions`
2. Включите режим разработчика
3. Выберите «Загрузить распакованное расширение»
4. Выберите папку проекта PlainTab

Режим веб-страницы:

Просто откройте `index.html` в браузере.

Не нужно устанавливать зависимости, не требуется сборка.

## Структура проекта

```text
PlainTab/
├── index.html              # Новая вкладка и вход в веб-версию
├── manifest.json           # Конфигурация браузерного расширения
├── css/                    # Стили страницы
├── js/                     # Логика страницы
├── js/wallpaper/           # Функции, связанные с обоями
├── _locales/               # Мультиязычность расширения
├── docs/                   # Документация
├── icon/                   # Иконки
└── imgs/                   # Скриншоты и материалы для магазина
```
