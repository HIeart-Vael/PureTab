## 🆕 Что нового в v3

v3 — это **полная переработка с нуля** с прорывом: **двухслойная система обоев с нулевым мерцанием**.

<details>
<summary><b>💡 Почему v2 мерцала?</b></summary>

Старая версия использовала один `<div>` с переключением CSS `background-image`. При переходе от миниатюры (правило stylesheet) к полному изображению (inline style) происходила смена каскада — в течение как минимум одного кадра отрисованный фон исчезал, обнажая серую подложку.

</details>

**Решение v3 — Двухслойная композиция:**
1. `#wallpaperBack` — всегда содержит видимое изображение. `preload.js` синхронно записывает миниатюру 640px до первой отрисовки браузера
2. `#wallpaperFront` — изначально прозрачен. После декодирования полного изображения плавно появляется сверху
3. Всегда хотя бы один слой содержит видимое изображение → **ноль серого мерцания**

Подробнее см. [V3_NOTE.md](./V3_NOTE.md).

## ✨ Почему PlainTab?

- 🔒 **Абсолютно чистая приватность** — Никакого сбора личных данных. Все обои хранятся локально
- 🚀 **Единый старт за одну минуту** — Установите как домашнюю страницу + установите расширение
- 🧩 **Настолько лёгкий, что вы его не чувствуете** — Ноль зависимостей, чистый JavaScript
- 🌍 **Работает из коробки** — Автоопределение языка браузера (16), поддержка Google / Bing / Baidu / DuckDuckGo

## 💡 Рекомендация разработчика: три обоев, три входа

Вы установили расширение — ваша новая вкладка уже выглядит отлично. Но вот чего вы, возможно, не знаете: PlainTab также развёрнут ещё в двух местах:

| Вход | Настройка | URL |
|------|--------|----------|
| 🧩 **Новая вкладка** | Расширение браузера | Загрузить это расширение |
| 🌐 **Стартовая страница** | Запуск браузера | `plaintab.netlify.app` |
| 🏠 **Домашняя страница** | Кнопка «Домой» | `kaininx.github.io/PlainTab` |

Установите `plaintab.netlify.app` как стартовую страницу браузера, пусть она следует за ежедневным обновлением Bing. Каждый раз, когда вы запускаете браузер, это ваши **вторые обои**.

Да, это ещё не всё. Найдите «Кнопку «Домой»» в настройках внешнего вида браузера, вставьте `kaininx.github.io/PlainTab`, выберите другие понравившиеся обои. Теперь у вас есть **третьи обои**.

Три входа полностью изолированы. Дайте каждому свои локальные обои или позвольте каждому следовать за ежедневным обновлением Bing. Запустите браузер: одни обои. Нажмите кнопку «Домой»: другие. Откройте новую вкладку: третьи. Ротация гарантирована.

**Настройка:**
1. Установите расширение → Новая вкладка ✓
2. Настройки браузера → При запуске → Открыть определённую страницу → `https://plaintab.netlify.app`
3. Настройки браузера → Внешний вид → Показывать кнопку «Домой» → `https://kaininx.github.io/PlainTab`

## 🛠️ Использование

| Действие | Эффект |
|----------|--------|
| Навести мышь на правый верхний угол | Показать значки языка / настроек |
| Навести мышь ближе к центру | Строка поиска появляется (режим Hover) |
| Нажать на шестерёнку | Открыть панель обоев и дополнительных опций |
| Нажать на глобус | Сменить язык интерфейса |
| Нажать на иконку поисковика | Цикл: Google → Bing → Baidu → DuckDuckGo |
| Нажать `Enter` в строке поиска | Поиск выбранным поисковиком |
| Нажать `Esc` | Закрыть все панели |

### Обои
- **Bing дня**: Автоматически загружается раз в день. Кэшируется только сегодняшнее изображение.
- **Локальные обои**: Загружайте изображения любого размера (IndexedDB, **без ограничений**). Сохраняется только последнее изображение. Сброс в один клик.

### Дополнительные опции
| Опция | Описание |
|-------|----------|
| Режим строки поиска | При наведении / Всегда / Скрыта |
| Прозрачность значка | 0 – 1 (по умолчанию 0.45) |
| Поисковая система | Google / Bing / Baidu / DuckDuckGo |

> **Расширение Chrome vs. Веб-версия — Разница в поиске:** Для соблюдения политики «Единого назначения» Chrome Web Store расширение использует Chrome Search API, который уважает поисковую систему по умолчанию, установленную в настройках браузера пользователя. Функция переключения поисковых систем недоступна в режиме расширения. Веб-версия (Netlify / GitHub Pages) не подпадает под это ограничение и сохраняет полный селектор поисковых систем. За исключением реализации поиска, обе версии функционально идентичны.

> Все настройки сохраняются в `localStorage`. Без аккаунта, без облачной синхронизации.

## 🔧 Нижняя панель на новой вкладке

После установки расширения Chrome / Edge показывает нижний колонтитул в правом нижнем углу страницы новой вкладки (с названием расширения). Это поведение браузера, а не то, что добавляет PlainTab.

**Как скрыть (из [Справки Chrome](https://support.google.com/chrome/answer/11032183?hl=ru)):**

Откройте новую вкладку → нажмите на значок «Настроить Chrome» ✏️ в правом нижнем углу → Нижний колонтитул → отключите «Показывать нижний колонтитул на странице новой вкладки».
