## 🆕 v3 में नया क्या है

v3 **शुरू से पूर्ण पुनर्लेखन** है जिसमें एक सफलता है: **शून्य-झिलमिलाहट दोहरी-परत वॉलपेपर प्रणाली**।

<details>
<summary><b>💡 v2 में झिलमिलाहट क्यों थी?</b></summary>

पुराने संस्करण में CSS `background-image` स्विचिंग के साथ एक एकल `<div>` का उपयोग किया गया था। थंबनेल (स्टाइलशीट नियम) से पूर्ण छवि (इनलाइन स्टाइल) पर जाने के लिए एक कैस्केड परिवर्तन की आवश्यकता थी — जिसके दौरान ब्राउज़र कम से कम एक फ्रेम के लिए रेंडर की गई पृष्ठभूमि को हटा देता था, जिससे ग्रे पृष्ठभूमि दिखाई देती थी।

</details>

**v3 का समाधान — दोहरी-परत संयोजन:**
1. `#wallpaperBack` — हमेशा एक दृश्यमान छवि रखता है। `preload.js` ब्राउज़र के पहले पेंट से पहले समकालिक रूप से 640px थंबनेल लिखता है
2. `#wallpaperFront` — पारदर्शी शुरू होता है। पूर्ण छवि डिकोड होने के बाद, ऊपर फीका हो जाता है
3. कम से कम एक परत में हमेशा एक दृश्यमान छवि होती है → **कोई ग्रे फ्लैश नहीं**

पूरी तकनीकी जानकारी के लिए [CHANGELOG.md](./CHANGELOG.md) देखें।

## ✨ PlainTab क्यों?

- 🔒 **पूरी तरह से साफ़ गोपनीयता** — कोई व्यक्तिगत डेटा एकत्र नहीं किया जाता। सभी वॉलपेपर स्थानीय रूप से संग्रहीत होते हैं।
- 🚀 **एक मिनट में एकीकृत ब्राउज़िंग प्रारंभ** — होमपेज के रूप में सेट करें + एक्सटेंशन इंस्टॉल करें। एक्सटेंशन कभी भी होमपेज बदलने के लिए बाध्य नहीं करता।
- 🧩 **इतना हल्का कि आपको महसूस भी न हो** — शून्य निर्भरताएँ, शुद्ध वेनिला जावास्क्रिप्ट, तत्काल प्रारंभ।
- 🌍 **बॉक्स से बाहर काम करता है** — ब्राउज़र भाषा (16) का स्वतः पता लगाता है, Google / Bing / Baidu / DuckDuckGo का समर्थन करता है।

## 🛠️ उपयोग

| क्रिया | प्रभाव |
|--------|--------|
| माउस को ऊपर-दाएँ कोने पर ले जाएँ | भाषा / सेटिंग आइकन दिखाएँ |
| माउस को केंद्र के पास ले जाएँ | खोज बार प्रकट होता है (होवर मोड) |
| गियर आइकन पर क्लिक करें | वॉलपेपर और उन्नत विकल्प पैनल खोलें |
| ग्लोब आइकन पर क्लिक करें | इंटरफ़ेस भाषा बदलें |
| खोज इंजन आइकन पर क्लिक करें | Google → Bing → Baidu → DuckDuckGo स्विच करें |
| खोज बार में `Enter` दबाएँ | वर्तमान इंजन से खोजें |
| `Esc` दबाएँ | सभी पैनल बंद करें |

### वॉलपेपर
- **बिंग दैनिक**: दिन में एक बार स्वचालित रूप से प्राप्त होता है। केवल आज की छवि स्थानीय रूप से कैश की जाती है।
- **स्थानीय वॉलपेपर**: किसी भी आकार की छवियाँ अपलोड करें (IndexedDB, **कोई फ़ाइल आकार सीमा नहीं**)। केवल अंतिम अपलोड की गई छवि रखी जाती है। एक क्लिक में बिंग दैनिक पर रीसेट करें।

### उन्नत विकल्प
| विकल्प | विवरण |
|--------|-------------|
| खोज बार मोड | होवर / हमेशा / छिपा हुआ |
| आइकन अपारदर्शिता | 0 – 1 (डिफ़ॉल्ट 0.45) |
| खोज इंजन | Google / Bing / Baidu / DuckDuckGo |

> सभी सेटिंग `localStorage` में सहेजी जाती हैं। कोई खाता नहीं, कोई क्लाउड सिंक नहीं।
