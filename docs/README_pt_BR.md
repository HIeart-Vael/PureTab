## O que é isso

PlainTab é uma nova guia para o navegador.

Ele faz algo bem simples: quando você abre uma nova guia, primeiro vê um belo papel de parede, e então pode pesquisar diretamente, digitar uma URL ou abrir seus links rápidos.

Ele não tem feed de notícias, nem tela cheia de widgets, nem clima, saudações ou listas de tarefas, e também não puxa sua atenção para outro lugar. É mais como uma mesa limpa: está lá quando você precisa, e não te incomoda quando não precisa.

## Por que vale a pena experimentar

### Imagem desde o primeiro instante

A sensação mais marcante do PlainTab é a velocidade.

Muitas novas guias mostram uma tela em branco antes da imagem aparecer. O PlainTab evita ao máximo essa sensação. Depois de usar uma vez, com o cache do navegador já presente, na próxima vez que abrir uma guia o papel de parede aparece quase instantaneamente, como se sempre estivesse ali.

Não é aquela "otimização de performance" que só existe nos parâmetros — é uma fluidez que você realmente sente.

### Confortável à primeira vista

Sua interface padrão é bem contida: papel de parede, barra de pesquisa, alguns botões pequenos.

Não te joga um monte de informação logo de cara, nem transforma a página inicial do navegador em outra estação de trabalho. Sua beleza vem da subtração, não do acúmulo.

### Muitas formas de brincar com o papel de parede

Você pode usar o papel de parede diário do Bing, enviar suas próprias imagens ou escolher uma pasta de papéis de parede do seu computador.

Se gosta de mexer nas coisas, também pode conectar feeds RSS de imagens ou usar sua própria API de imagens. Dá pra usar de forma simples, mas também tem espaço para explorar a fundo.

### A pesquisa é muito natural

A barra de pesquisa pode ser colocada na posição que você preferir, e você pode ajustar tamanho, bordas arredondadas, transparência e modo de exibição.

Você pode deixá-la sempre visível, ou escondida e só aparecer quando precisar. Parece leve, mas no dia a dia é bem prática.

### Links rápidos sem ocupar a página inicial

Muitas novas guias enchem a tela de links rápidos. O PlainTab não faz isso.

Ele coloca os links rápidos dentro da paleta de comandos. Você pode pesquisar, adicionar, editar, importar favoritos e até guardar alguns links em um espaço oculto. A página inicial continua limpa, mas sem perder funcionalidades.

### Configurações de sobra, mas sem barulho

O PlainTab permite ajustar papel de parede, barra de pesquisa, atalhos de teclado, backup, idioma e detalhes da interface.

Mas essas configurações não ficam empilhadas na página inicial. No dia a dia você vê uma nova guia tranquila; quando quiser ajustar algo, é só abrir o painel de configurações.

## Este é um projeto colaborativo com IA

O PlainTab também é ótimo para aprender.

Não é um projeto de brinquedo que só serve de demonstração — é uma extensão de nova guia completa: com interface real, configurações de usuário, persistência de dados, importação e exportação, múltiplos idiomas, e suporte tanto ao ambiente de extensão do navegador quanto ao de página web comum.

Este projeto foi desenvolvido com colaboração intensiva de IA no desenvolvimento, na organização, na documentação e nas iterações. Você pode usá-lo como um ótimo material de estudo:

- Para ver como a IA participa do desenvolvimento de um projeto real.
- Para aprender como se faz uma extensão de navegador.
- Para começar no frontend sem encarar frameworks complexos de cara.
- Para estudar um projeto pequeno mas bem completo, da ideia às funcionalidades, experiência e documentação.

Seu código não depende de ferramentas de build complexas — abra os arquivos e leia, abra a página e ela funciona. Para iniciantes, isso é muito mais amigável do que instalar um monte de dependências logo de início.

## Para quem é

O PlainTab é para quem:

- Quer uma nova guia mais bonita.
- Não gosta de ver um monte de informação ao abrir o navegador.
- Gosta de papéis de parede diários ou tem sua própria coleção.
- Quer uma página inicial leve mas com funcionalidades completas.
- Busca um projeto de colaboração com IA que seja real, completo e bom para aprender.
- Quer começar com extensões de navegador ou desenvolvimento frontend.

## O que não vai fazer

O PlainTab vai continuar sendo contido. Algumas funções, por mais comuns que sejam, não necessariamente cabem neste projeto.

Muito provavelmente nunca terá:

- Feed de notícias, trending topics, conteúdo recomendado.
- Anúncios ao abrir, cards promocionais, espaços patrocinados.
- Grandes painéis de clima, calendário ou tarefas.
- Reprodução automática de vídeos ou fundos animados.
- Sistema de contas, funções sociais, fluxo de conteúdo na nuvem.
- A página inicial lotada de dezenas de atalhos.
- Versão para Safari. Exige um custo extra de publicação e manutenção que, para um projeto pessoal como este, ainda não é realista.

Não é que essas funções sejam ruins — é que elas transformariam o PlainTab em outra coisa.

Se quando você abre uma nova guia só quer um momento de calma, pesquisar algo ou olhar um papel de parede, o PlainTab está do seu lado.

## O que pode vir depois

Eu gostaria que o PlainTab continuasse crescendo nestas direções:

- Fontes de papel de parede mais estáveis.
- Uma experiência de configuração mais confortável.
- Documentação de aprendizado mais fácil de entender.
- Explicações do código mais amigáveis para iniciantes.
- Um registro mais completo do desenvolvimento colaborativo com IA.
- Possível suporte ao Firefox.

O suporte ao Firefox ainda não dá para prometer, porque o ambiente de extensões, as permissões e o comportamento das APIs podem ser diferentes do Chrome e do Edge. Mas se as condições forem favoráveis no futuro, eu gostaria que o PlainTab não ficasse só nos navegadores baseados em Chromium.

Se você se interessa por extensões para Firefox, compatibilidade entre navegadores ou desenvolvimento assistido por IA, também é bem-vindo para explorar junto. Este projeto é perfeito para aprender enquanto usa.

## Visão geral das funcionalidades

| Funcionalidade | Descrição |
|------|------|
| Substituição da nova guia | Assume a nova guia do navegador após a instalação |
| Página inicial online | Também funciona diretamente como página web |
| Papel de parede diário | Compatível com o papel de parede diário do Bing |
| Papéis de parede locais | Permite enviar imagens e escolher pastas locais |
| Papéis de parede RSS / API | É possível conectar mais fontes de imagens |
| Barra de pesquisa | Posição, tamanho, estilo e modo de exibição ajustáveis |
| Histórico de pesquisa | Salva pesquisas recentes ou pode ser desativado |
| Links rápidos | Gerenciados pela paleta de comandos, sem ocupar a página inicial |
| Espaço oculto | Guarda separadamente os links que você não quer exibir |
| Painel de configurações | Ajusta interface, papéis de parede, atalhos e dados |
| Backup de dados | Permite exportar, importar e fazer backup criptografado |
| Múltiplos idiomas | Interface multilíngue |
| Colaboração com IA | Ótimo para aprender como a IA participa do desenvolvimento de projetos |

## Início rápido

### Instalar a extensão

[Instalar o PlainTab na Chrome Web Store](https://chromewebstore.google.com/detail/plaintab-%C2%B7-minimal-new-ta/jhpfjcefcmooplmaimgdafohdlhacjdo)

### Testar online

[plaintab.kaininx.workers.dev](https://plaintab.kaininx.workers.dev)

### Executar localmente

```bash
git clone https://github.com/kaininx/PlainTab.git
```

Modo extensão:

1. Abra `chrome://extensions`
2. Ative o modo desenvolvedor
3. Selecione "Carregar extensão descompactada"
4. Escolha a pasta do projeto PlainTab

Modo página web:

Abra o `index.html` diretamente no navegador.

Não precisa instalar dependências nem fazer build.

## Estrutura do projeto

```text
PlainTab/
├── index.html              # Entrada da nova guia e página web
├── manifest.json           # Configuração da extensão do navegador
├── css/                    # Estilos da página
├── js/                     # Lógica da página
├── js/wallpaper/           # Funções relacionadas ao papel de parede
├── _locales/               # Múltiplos idiomas da extensão
├── docs/                   # Documentação
├── icon/                   # Ícones
└── imgs/                   # Capturas de tela e recursos para a loja
```
