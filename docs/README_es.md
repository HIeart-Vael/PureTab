## ¿Qué es esto?

PlainTab es una nueva pestaña para el navegador.

Hace algo muy simple: cuando abres una nueva pestaña, ves primero un fondo de pantalla bonito, y luego puedes buscar directamente, escribir una URL o abrir tus enlaces rápidos.

No tiene flujo de noticias, ni pantalla llena de widgets, ni clima, ni saludos, ni listas de tareas, ni nada que te distraiga. Es más como un escritorio limpio: está ahí cuando lo necesitas, y no te molesta cuando no.

## Por qué vale la pena probarlo

### Imagen desde el primer instante

La sensación más evidente con PlainTab es la velocidad.

Muchas nuevas pestañas muestran un espacio en blanco antes de que aparezca la imagen. PlainTab evita esa sensación todo lo posible. Después de usarlo una vez, con la caché del navegador ya presente, la siguiente vez que abras una pestaña el fondo aparecerá casi al instante, como si siempre hubiera estado ahí.

No es una "optimización de rendimiento" teórica, sino una fluidez que puedes sentir de verdad.

### Cómodo a primera vista

Su interfaz por defecto es muy contenida: fondo de pantalla, barra de búsqueda, unos pocos botones pequeños.

No te bombardea con información al abrirlo, ni convierte la página de inicio del navegador en otro centro de trabajo. Su belleza viene de lo que quita, no de lo que añade.

### Muchas formas de jugar con el fondo

Puedes usar el fondo diario de Bing, subir tus propias imágenes o elegir una carpeta de fondos de tu ordenador.

Si te gusta experimentar, también puedes conectar fuentes RSS de imágenes o tu propia API de imágenes. Es fácil para un uso simple, y hay espacio para profundizar si quieres.

### La búsqueda se siente natural

La barra de búsqueda se puede colocar donde prefieras, y puedes ajustar su tamaño, bordes redondeados, transparencia y modo de visualización.

Puedes tenerla siempre visible, u oculta normalmente y que solo aparezca cuando la necesites. Parece ligera, pero en el día a día resulta muy práctica.

### Enlaces rápidos sin ocupar la página principal

Muchas nuevas pestañas llenan la pantalla de enlaces rápidos. PlainTab no hace eso.

Coloca los enlaces rápidos dentro de la paleta de comandos. Puedes buscar, añadir, editar, importar marcadores, e incluso guardar algunos enlaces en un espacio oculto. La página principal sigue limpia, pero sin perder funcionalidad.

### Ajustes de sobra, pero sin ruido

PlainTab te permite ajustar el fondo, la barra de búsqueda, los atajos de teclado, las copias de seguridad, el idioma y los detalles de la interfaz.

Pero esos ajustes no se amontonan en la página principal. Lo que ves a diario es una nueva pestaña tranquila; cuando quieras ajustar algo, abres el panel de configuración.

## Un proyecto de colaboración con IA

PlainTab también es ideal para aprender.

No es un proyecto de juguete que solo funciona en demo, sino una extensión de nueva pestaña completa: con interfaz real, configuración de usuario, persistencia de datos, importación y exportación, múltiples idiomas, y soporte tanto para el entorno de extensión del navegador como para página web normal.

Este proyecto se ha desarrollado con una colaboración intensiva de IA en el desarrollo, la organización, la documentación y las iteraciones. Puedes tomarlo como un excelente material de aprendizaje:

- Para ver cómo la IA participa en el desarrollo de un proyecto real.
- Para aprender cómo se construye una extensión de navegador.
- Para empezar con el desarrollo frontend sin enfrentarte de golpe a frameworks complejos.
- Para estudiar un proyecto pequeño pero completo, desde la idea hasta las funcionalidades, la experiencia y la documentación.

Su código no depende de herramientas de build complejas: abres los archivos y los lees, abres la página y funciona. Para principiantes, esto es mucho más amigable que instalar un montón de dependencias desde el principio.

## Para quién es

PlainTab es para quienes:

- Quieren una nueva pestaña más bonita.
- No les gusta ver un montón de información al abrir el navegador.
- Disfrutan de los fondos diarios o tienen su propia colección de fondos.
- Quieren una página de inicio ligera pero completa en funciones.
- Buscan un proyecto de colaboración con IA que sea real, completo y apto para aprender.
- Quieren iniciarse en extensiones de navegador o desarrollo frontend.

## Lo que no hará

PlainTab seguirá siendo contenido. Algunas funciones, por muy comunes que sean, no encajan necesariamente en este proyecto.

Muy probablemente nunca tendrá:

- Flujo de noticias, tendencias, contenido recomendado.
- Anuncios al abrir, tarjetas promocionales, espacios patrocinados.
- Grandes paneles de clima, calendario o tareas.
- Reproducción automática de vídeos o fondos animados.
- Sistema de cuentas, funciones sociales, flujo de contenido en la nube.
- La página principal llena de decenas de accesos directos.
- Versión para Safari. Requiere un coste adicional de publicación y mantenimiento que, para un proyecto personal como este, no es realista por ahora.

No es que esas funciones sean malas, sino que convertirían PlainTab en otra cosa.

Si cuando abres una nueva pestaña solo quieres un momento de calma, buscar algo o mirar un fondo bonito, PlainTab está de tu lado.

## Lo que podría venir después

Me gustaría que PlainTab siguiera creciendo en estas direcciones:

- Fuentes de fondos más estables.
- Una experiencia de configuración más cómoda.
- Documentación de aprendizaje más fácil de entender.
- Explicaciones del código más amigables para principiantes.
- Un registro más completo del desarrollo colaborativo con IA.
- Posible compatibilidad con Firefox.

El soporte para Firefox no se puede prometer ahora mismo, porque el entorno de extensiones, los permisos y el comportamiento de las API pueden ser diferentes a los de Chrome y Edge. Pero si las condiciones son adecuadas en el futuro, me gustaría que PlainTab no se quedara solo en los navegadores basados en Chromium.

Si te interesan las extensiones de Firefox, la compatibilidad entre navegadores o el desarrollo asistido por IA, también eres bienvenido a explorar. Este proyecto es perfecto para aprender mientras se usa.

## Vista general de funciones

| Función | Descripción |
|------|------|
| Reemplazo de nueva pestaña | Sustituye la nueva pestaña del navegador al instalar |
| Página de inicio en línea | También funciona directamente como página web |
| Fondo diario | Compatible con el fondo diario de Bing |
| Fondos locales | Permite subir imágenes y elegir carpetas locales |
| Fondos RSS / API | Se pueden conectar más fuentes de imágenes |
| Barra de búsqueda | Posición, tamaño, estilo y modo de visualización ajustables |
| Historial de búsqueda | Guarda las búsquedas recientes o se puede desactivar |
| Enlaces rápidos | Gestionados desde la paleta de comandos, sin ocupar la página principal |
| Espacio oculto | Guarda por separado los enlaces que no quieres mostrar |
| Panel de configuración | Ajusta la interfaz, los fondos, los atajos y los datos |
| Copia de seguridad | Permite exportar, importar y hacer copias cifradas |
| Múltiples idiomas | Interfaz multilingüe |
| Colaboración con IA | Ideal para aprender cómo la IA participa en el desarrollo de proyectos |

## Inicio rápido

### Instalar la extensión

[Instalar PlainTab en Chrome Web Store](https://chromewebstore.google.com/detail/plaintab-%C2%B7-minimal-new-ta/jhpfjcefcmooplmaimgdafohdlhacjdo)

### Probar en línea

[plaintab.kaininx.workers.dev](https://plaintab.kaininx.workers.dev)

### Ejecutar en local

```bash
git clone https://github.com/kaininx/PlainTab.git
```

Modo extensión:

1. Abre `chrome://extensions`
2. Activa el modo desarrollador
3. Selecciona "Cargar extensión descomprimida"
4. Elige la carpeta del proyecto PlainTab

Modo página web:

Abre `index.html` directamente en el navegador.

No necesitas instalar dependencias ni hacer build.

## Estructura del proyecto

```text
PlainTab/
├── index.html              # Entrada de nueva pestaña y página web
├── manifest.json           # Configuración de la extensión del navegador
├── css/                    # Estilos de la página
├── js/                     # Lógica de la página
├── js/wallpaper/           # Funciones relacionadas con el fondo
├── _locales/               # Múltiples idiomas de la extensión
├── docs/                   # Documentación
├── icon/                   # Iconos
└── imgs/                   # Capturas de pantalla y recursos para la tienda
```
