# PlainTab GitHub Release Body

---

**PlainTab v3.2.0**

- Install from the [Chrome Web Store](https://chromewebstore.google.com/detail/plaintab-%C2%B7-minimal-new-ta/jhpfjcefcmooplmaimgdafohdlhacjdo) for automatic updates.
- Or download the `.crx` file below and drag it into `chrome://extensions`.

**Highlights**

- Added the command palette for managing shortcuts without filling the homepage.
- Added more wallpaper sources: RSS feeds, custom APIs, and local wallpaper folders.
- Added WASM-powered wallpaper theme colors for a more cohesive visual experience.
- Added configuration import/export and search history.
- Migrated data to the v3.2 storage model for the new wallpaper and settings capabilities.
- Tightened extension permissions and search fallback behavior.
- Optimized startup, wallpaper rendering, gallery, panel animations, blur performance, and settings loading.
- Reorganized the code into clearer modules and refreshed the project documentation.

**Summary**

v3.2.0 is a substantial feature and performance release. PlainTab now supports command-palette shortcut management, RSS/API/local-folder wallpaper sources, WASM-based wallpaper theme colors, configuration import/export, and search history. Under the hood, the storage layer has moved to the v3.2 model, permissions and search fallback behavior are more restrained, and the startup, wallpaper, gallery, and settings paths have been modularized and optimized. The runtime framework remains lightweight and unchanged, so this is a minor-version upgrade rather than a breaking major release, but the everyday experience is significantly more complete.

**PlainTab v3.2.0**

- 前往 [Chrome 网上应用店](https://chromewebstore.google.com/detail/plaintab-%C2%B7-minimal-new-ta/jhpfjcefcmooplmaimgdafohdlhacjdo) 安装，可自动更新。
- 或下载下方 `.crx` 文件，拖入 `chrome://extensions` 页面即可。

**更新重点**

- 新增命令面板，用于管理快捷链接，主页不再被快捷方式铺满。
- 新增更多壁纸来源：RSS 图片源、自定义 API、本地壁纸文件夹。
- 新增 WASM 壁纸主题色能力，让界面色彩更贴合当前壁纸。
- 新增配置导入导出与搜索历史。
- 将数据层迁移到 v3.2 存储模型，支撑新的壁纸和设置能力。
- 收束扩展权限与搜索回退行为。
- 优化启动链路、壁纸渲染、画廊、面板动效、背景模糊性能和设置加载。
- 拆分并整理代码模块，同时刷新项目文档体系。

**总结**

v3.2.0 是一次大型功能与性能更新。PlainTab 新增了命令面板、RSS/API/本地文件夹壁纸、WASM 壁纸主题色、配置导入导出和搜索历史；底层迁移到 v3.2 存储模型，并进一步收束权限与搜索回退行为。启动、壁纸、画廊和设置链路也完成了模块化和性能优化。虽然运行框架仍保持轻量、没有破坏性变化，所以这是一次次版本号升级，但日常使用体验已经明显更完整。
