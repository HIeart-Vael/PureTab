변경 내역

• v3.1.3 · localStorage와 IndexedDB를 위한 스토리지 버전 마이그레이션 구현, 스토리지 키 이름 통일, 로컬 모드에서 썸네일 지속성 로직 수정.

• v3.1.2 · Chrome Web Store 규정 준수를 위해 _locales/pt를 pt_BR로 이름 변경; 변경 로그 형식을 간결한 스타일로 통일; 포르투갈어 파일을 pt_BR로 일관성 있게 이름 변경.

• v3.1.1 · Bing API를 Promise.any 경쟁 모드로 전환하고 8초 AbortController 타임아웃 추가. 전체 16개 언어 README를 새로운 가져오기 전략에 맞게 업데이트.

• v3.1.0 · 로컬 배경화면 갤러리: 여러 배경화면 업로드/삭제/순환, 일괄 파일 선택. 코드 리팩토링, 패널 스타일 개선, 다국어 현지화 강화. JSON API 도메인을 CSP에서 제거하여 수정.

• v3.0.5 · Bing API 프록시 업그레이드 (지역화 요청 + 이미지 중복 제거). 배경화면 캐시 최적화, localStorage 키 이름 통일. 16개 언어 체인지로그 추가. 로깅 및 코드 의미론 개선.

• v3.0.4 · 배경화면 로딩 실패 수정: CSP img-src 허용 목록에 Bing CDN 도메인이 누락되어 프록시 API 리디렉션 후 이미지가 브라우저에서 차단되는 문제 해결.

• v3.0.3 · CSSOM 우회로 첫 프레임 렌더링 가속화, CSP 보안 강화, 데드 코드 정리, SVG 아이콘 및 프로모션 자료 추가.

• v3.0.2 · 다크 모드 적응형 배경, 코드 구조 리팩토링, 하단 다국어 정보 표시줄.

• v3.0.1 · Chrome Search API 통합, Chrome 웹 스토어 단일 목적 정책 준수 심사 통과.

• v3.0.0 · 완전 재작성: 이중 레이어 아키텍처로 새 탭 흰색 화면 플래시 제거, 폴백 회색 배경 추가.

• v2.2.7 · 영/일/한/노/간체 5개 언어 README 추가, 기본 검색 모드를 '항상 표시'로 변경.

• v2.2.6 · 배경화면 로딩 로직 및 썸네일 캐시 최적화, Bing API 상수 정리.

• v2.2.5 · 배경화면 사전 로딩 및 첫 프레임 최적화로 새 탭 흰 화면 제거.

• v2.2.4 · 배경화면 저장소를 localStorage에서 IndexedDB로 마이그레이션.

• v2.2.3 · 로컬 배경화면 업로드, 검색 엔진 전환 경험 개선.

• v1.2.0 · 최초 릴리스: 배경화면 시스템, 16개 언어 UI, Chrome i18n 현지화, MIT 라이선스.
