## ما هذا

PlainTab هو صفحة علامة تبويب جديدة للمتصفح.

ما يفعله بسيط: عندما تفتح علامة تبويب جديدة، ترى أولاً خلفية جميلة، ثم تبحث مباشرة، أو تكتب عنوان URL، أو تفتح روابطك المختصرة.

ليس فيه تدفق أخبار، ولا شاشة مليئة بالودجات، ولا طقس، ولا تحية، ولا قائمة مهام، ولا يسحب انتباهك إلى مكان آخر. إنه أشبه بسطح مكتب نظيف: حين تحتاجه، يكون هناك؛ وحين لا تحتاجه، لا يزعجك.

## لماذا يستحق التجربة

### الصورة جاهزة بمجرد الفتح

أوضح إحساس يمنحه PlainTab هو السرعة.

العديد من صفحات علامات التبويب الجديدة تعرض وميضاً أبيض فارغاً عند الفتح، ثم تظهر الصورة بعد قليل. PlainTab يتجنب هذا الإحساس قدر الإمكان. بمجرد أن تستخدمه مرة، ويصبح لدى المتصفح ذاكرة مخبأة، ستظهر الخلفية بسرعة كبيرة في المرة التالية، وكأنها كانت هناك دائماً.

هذا ليس "تحسين أداء" مكتوباً في المعاملات، بل سلاسة تشعر بها مباشرة.

### مريح من أول نظرة

واجهته الافتراضية متزنة للغاية: خلفية، شريط بحث، وبضعة أزرار صغيرة.

لا يقذفك بوابل من المعلومات بمجرد الفتح، ولا يحول صفحة المتصفح الرئيسية إلى محطة عمل أخرى. جماله يأتي من القلة، لا من الكثرة.

### طرق متعددة للخلفيات

يمكنك استخدام خلفية Bing اليومية مباشرة، أو رفع صورك الخاصة، أو اختيار مجلد خلفيات من جهاز الكمبيوتر.

إذا كنت تحب التعديل، يمكنك أيضاً إضافة مصادر صور RSS، أو توصيل API صور خاص بك. الاستخدام البسيط ممكن، والتعمق متاح أيضاً.

### البحث سلس للغاية

يمكن وضع شريط البحث في المكان الذي تحبه، مع إمكانية تعديل الحجم، والزوايا الدائرية، والشفافية، وطريقة العرض.

يمكنك جعله مرئياً دائماً، أو مخفياً عادةً ولا يظهر إلا عند الحاجة. يبدو خفيفاً جداً، لكنه مريح جداً في الاستخدام اليومي.

### الروابط المختصرة لا تشغل الصفحة الرئيسية

العديد من علامات التبويب الجديدة تملأ الشاشة بالروابط المختصرة. PlainTab لا يفعل ذلك.

إنه يضع الروابط المختصرة داخل لوحة الأوامر. يمكنك البحث، والإضافة، والتعديل، واستيراد الإشارات المرجعية، ويمكنك أيضاً وضع بعض الروابط في مساحة مخفية. تبقى الصفحة الرئيسية نظيفة، لكن الوظائف لم تنقص.

### إعدادات كافية، بدون ضوضاء

يمكن لـ PlainTab ضبط الخلفيات، وشريط البحث، واختصارات لوحة المفاتيح، والنسخ الاحتياطي، واللغة، وتفاصيل الواجهة.

لكن هذه الإعدادات لا تتراكم على الصفحة الرئيسية. ما تراه عادةً هو علامة تبويب جديدة هادئة؛ وعند الحاجة للتعديل، تفتح لوحة الإعدادات.

## هذا مشروع بتعاون الذكاء الاصطناعي

PlainTab مناسب جداً للتعلم أيضاً.

إنه ليس مشروع لعبة على مستوى العرض فقط، بل إضافة علامة تبويب جديدة كاملة: بواجهة حقيقية، وإعدادات مستخدم، وحفظ بيانات، واستيراد وتصدير، وتعدد لغات، وبيئة إضافة متصفح، وبيئة صفحة ويب عادية.

استخدم هذا المشروع تعاون الذكاء الاصطناعي بشكل كبير في التطوير والتنظيم والتوثيق والتكرار. يمكنك اعتباره عينة تعلم جيدة:

- لترى كيف يشارك الذكاء الاصطناعي في تطوير مشروع حقيقي.
- لتتعلم كيف تُصنع إضافة متصفح.
- للبدء في الواجهات الأمامية دون مواجهة أطر عمل معقدة من البداية.
- لدراسة مشروع صغير مكتمل نسبياً من الفكرة إلى الوظائف والتجربة والتوثيق.

كوده لا يعتمد على أدوات بناء معقدة — افتح الملف لتقرأه، وافتح الصفحة لتشغلها. بالنسبة للمبتدئين، هذا أكثر وداً من تثبيت كومة من الاعتماديات في البداية.

## لمن يناسب

PlainTab يناسب هؤلاء الأشخاص:

- من يريد تغيير صفحة علامة تبويب جديدة إلى واحدة أجمل.
- من لا يحب رؤية وابل من المعلومات عند فتح المتصفح.
- من يحب الخلفيات اليومية، أو لديه مجموعة خلفيات خاصة.
- من يريد صفحة بداية خفيفة لكن كاملة الوظائف.
- من يبحث عن مشروع تعاون ذكاء اصطناعي حقيقي، كامل، ومناسب للتعلم.
- من يريد البدء في إضافات المتصفح أو مشاريع الواجهات الأمامية.

## ما لن يفعله

سيستمر PlainTab في الحفاظ على توازنه. بعض الميزات، حتى لو كانت مألوفة لكثيرين، قد لا تكون مناسبة لهذا المشروع.

على الأرجح لن يدعم:

- تدفق الأخبار، قوائم الترند، المحتوى الموصى به.
- إعلانات الشاشة الافتتاحية، بطاقات ترويجية، مساحات دعائية.
- لوحات طقس، تقويم، أو مهام كبيرة.
- تشغيل تلقائي للفيديو أو خلفيات متحركة.
- نظام حسابات، ميزات اجتماعية، تدفق محتوى سحابي.
- ملء الصفحة الرئيسية بعشرات الاختصارات.
- إصدار Safari. يتطلب تكلفة نشر وصيانة إضافية، وهو غير واقعي حالياً لمشروع شخصي.

ليس لأن هذه الميزات سيئة، بل لأنها ستحول PlainTab إلى شيء آخر.

إذا كنت تفتح علامة تبويب جديدة فقط للهدوء، أو للبحث عن شيء، أو للنظر إلى خلفية، فإن PlainTab يقف في صفك.

## ما قد يفعله لاحقاً

أرغب في أن يستمر PlainTab في النمو نحو هذه الاتجاهات:

- مصادر خلفيات أكثر استقراراً.
- تجربة إعدادات أكثر راحة.
- وثائق تعلم أسهل فهماً.
- شرح كود أكثر ملاءمة للمبتدئين.
- سجل تطوير أكثر اكتمالاً بتعاون الذكاء الاصطناعي.
- دعم Firefox محتمل.

لا يمكن الوعد بدعم Firefox مباشرة الآن، لأن بيئة إضافة المتصفح، والصلاحيات، وسلوك API قد تختلف عن Chrome / Edge. لكن إذا كانت الظروف مناسبة في المستقبل، أتمنى ألا يبقى PlainTab محصوراً في متصفحات Chromium فقط.

إذا كنت مهتماً بإضافات Firefox، أو توافق المتصفحات، أو التطوير بمساعدة الذكاء الاصطناعي، فنرحب باستكشاف ذلك معاً. هذا المشروع مناسب جداً للتعلم أثناء الاستخدام.

## نظرة على الميزات

| الميزة | الوصف |
|------|------|
| استبدال علامة التبويب الجديدة | يتولى صفحة علامة التبويب الجديدة في المتصفح بعد التثبيت |
| صفحة بداية على الإنترنت | يمكن استخدامها مباشرة كصفحة ويب |
| خلفية يومية | يدعم خلفية Bing اليومية |
| خلفيات محلية | يدعم رفع الصور واختيار مجلدات محلية |
| خلفيات RSS / API | يمكن إضافة المزيد من مصادر الصور |
| شريط البحث | قابل لتعديل الموضع والحجم والنمط وطريقة العرض |
| سجل البحث | يمكن حفظ عمليات البحث الأخيرة، أو إيقافها |
| روابط مختصرة | تدار عبر لوحة الأوامر، لا تشغل الصفحة الرئيسية |
| مساحة مخفية | حفظ منفصل للروابط التي لا ترغب في عرضها |
| لوحة الإعدادات | ضبط الواجهة والخلفيات والاختصارات والبيانات |
| نسخ احتياطي للبيانات | يدعم التصدير والاستيراد والنسخ الاحتياطي المشفر |
| تعدد اللغات | يدعم واجهة متعددة اللغات |
| تعاون الذكاء الاصطناعي | مناسب للتعلم عن مشاركة الذكاء الاصطناعي في تطوير المشاريع |

## بداية سريعة

### تثبيت الإضافة

[ثبّت PlainTab من Chrome Web Store](https://chromewebstore.google.com/detail/plaintab-%C2%B7-minimal-new-ta/jhpfjcefcmooplmaimgdafohdlhacjdo)

### تجربة عبر الإنترنت

[plaintab.kaininx.workers.dev](https://plaintab.kaininx.workers.dev)

### التشغيل المحلي

```bash
git clone https://github.com/kaininx/PlainTab.git
```

وضع الإضافة:

1. افتح `chrome://extensions`
2. فعّل وضع المطور
3. اختر "تحميل إضافة غير مضغوطة"
4. اختر مجلد مشروع PlainTab

وضع صفحة الويب:

افتح `index.html` مباشرة في المتصفح.

لا حاجة لتثبيت اعتماديات أو بناء.

## هيكل المشروع

```text
PlainTab/
├── index.html              # مدخل علامة التبويب الجديدة وصفحة الويب
├── manifest.json           # إعدادات إضافة المتصفح
├── css/                    # أنماط الصفحة
├── js/                     # منطق الصفحة
├── js/wallpaper/           # وظائف متعلقة بالخلفيات
├── _locales/               # تعدد لغات الإضافة
├── docs/                   # الوثائق
├── icon/                   # الأيقونات
└── imgs/                   # لقطات الشاشة ومواد المتجر
```
