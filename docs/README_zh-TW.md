## 這是什麼

PlainTab 是一個瀏覽器新分頁。

它做的事情很簡單：當你打開新分頁時，先看到一張漂亮的桌布，然後直接搜尋、輸入網址，或者打開自己的快捷連結。

它沒有新聞流，沒有滿螢幕小工具，沒有天氣、問候語、待辦清單，也不會把你的注意力往別的地方拉。它更像一張乾淨的桌面：你需要的時候，它在那裡；你不需要的時候，它不打擾你。

## 為什麼值得試試

### 打開就有畫面

PlainTab 最明顯的感覺是快。

很多新分頁打開時，會先閃出一片空白，等一下圖片才出來。PlainTab 盡量避免這種感覺。只要你用過一次、瀏覽器裡已經有快取，下次打開時桌布會非常快地出現，像是一直就在那兒。

這不是那種寫在參數裡的「效能最佳化」，而是你能直接感覺到的流暢。

### 第一眼很舒服

它的預設介面很克制：桌布、搜尋列、幾個小按鈕。

不會一打開就塞給你一堆資訊，也不會把瀏覽器首頁變成另一個工作臺。它的美感來自少，而不是來自堆。

### 桌布玩法很多

你可以直接用 Bing 每日桌布，也可以上傳自己的圖片，或者選擇電腦裡的桌布資料夾。

如果你喜歡折騰，也可以接 RSS 圖片來源，或者接自己的圖片 API。簡單用可以很簡單，想深入玩也有空間。

### 搜尋很順手

搜尋列可以放在你喜歡的位置，也可以調整大小、圓角、透明度和顯示方式。

你可以讓它一直顯示，也可以讓它平時隱藏，只在需要時出現。它看起來很輕，但日常用起來很方便。

### 快捷連結不佔首頁

很多新分頁會把快捷連結鋪滿螢幕。PlainTab 沒有這麼做。

它把快捷連結放進命令面板裡。你可以搜尋、新增、編輯、匯入書籤，也可以把一些連結放進隱藏空間。首頁還是乾淨的，但功能沒有少。

### 設定夠多，但不吵

PlainTab 可以調桌布、搜尋列、快捷鍵、備份、語言、介面細節。

但這些設定不會堆在首頁上。平時你看到的是一個安靜的新分頁；需要調整時，再打開設定面板。

## 這是一個 AI 協作專案

PlainTab 也很適合拿來學習。

它不是一個只停留在展示層面的玩具專案，而是一個完整的新分頁擴充功能：有真實介面，有使用者設定，有資料儲存，有匯入匯出，有多語言，有瀏覽器擴充環境，也有普通網頁環境。

這個專案在開發、整理、文件與迭代過程中大量使用 AI 協作完成。你可以把它當作一個很好的學習樣本：

- 想看看 AI 怎麼參與真實專案開發。
- 想學習一個瀏覽器擴充功能是怎麼做出來的。
- 想入門前端，但不想一上來就面對複雜框架。
- 想研究一個從想法、功能、體驗到文件都比較完整的小專案。

它的程式碼不依賴複雜建置工具，打開檔案就能看，打開頁面就能跑。對新手來說，這比一上來安裝一堆依賴更友善。

## 適合誰

PlainTab 適合這些人：

- 想換一個更漂亮的新分頁。
- 不喜歡打開瀏覽器就看到一堆資訊。
- 喜歡每日桌布，或者有自己的桌布收藏。
- 想要一個輕量但功能完整的起始頁。
- 想找一個真實、完整、適合學習的 AI 協作專案。
- 想入門瀏覽器擴充功能或前端專案。

## 不會做什麼

PlainTab 會繼續保持克制。有些功能就算很多人熟悉，也不一定適合放進這個專案。

大概不會支援：

- 新聞流、熱榜、推薦內容。
- 開螢幕廣告、推廣卡片、贊助位。
- 大面積天氣、日曆、待辦面板。
- 自動播放影片或動態背景。
- 帳號系統、社交功能、雲端內容流。
- 把首頁鋪滿幾十個捷徑。
- Safari 版本。它需要額外的發布和維護成本，對現在這個個人專案來說還不太現實。

不是這些功能不好，而是它們會把 PlainTab 變成另一個東西。

如果你打開新分頁只是想安靜一下、搜個東西、看一眼桌布，那 PlainTab 應該站在你這邊。

## 以後可能會做什麼

我比較想讓 PlainTab 繼續往這幾個方向長：

- 更穩定的桌布來源。
- 更舒服的設定體驗。
- 更容易理解的學習文件。
- 更適合新手閱讀的程式碼說明。
- 更完整的 AI 協作開發記錄。
- 可能支援 Firefox。

Firefox 支援現在還不能直接承諾，因為瀏覽器擴充環境、權限、API 行為都可能跟 Chrome / Edge 不一樣。但如果未來條件合適，我會希望 PlainTab 不只停在 Chromium 系瀏覽器裡。

如果你正好對 Firefox 擴充功能、瀏覽器相容或 AI 輔助開發感興趣，也歡迎一起研究。這個專案很適合邊用邊學。

## 功能一覽

| 功能 | 說明 |
|------|------|
| 新分頁取代 | 安裝後接管瀏覽器新分頁 |
| 線上起始頁 | 也可以直接作為網頁使用 |
| 每日桌布 | 支援 Bing 每日桌布 |
| 本機桌布 | 支援上傳圖片和選擇本機資料夾 |
| RSS / API 桌布 | 可以接入更多圖片來源 |
| 搜尋列 | 可調整位置、大小、樣式和顯示方式 |
| 搜尋歷史 | 可儲存最近搜尋，也可以關閉 |
| 快捷連結 | 用命令面板管理，不佔首頁 |
| 隱藏空間 | 單獨儲存不想展示的連結 |
| 設定面板 | 調整介面、桌布、快捷鍵和資料 |
| 資料備份 | 支援匯出、匯入和加密備份 |
| 多語言 | 支援多語言介面 |
| AI 協作 | 適合用來學習 AI 參與專案開發 |

## 快速開始

### 安裝擴充功能

[在 Chrome Web Store 安裝 PlainTab](https://chromewebstore.google.com/detail/plaintab-%C2%B7-minimal-new-ta/jhpfjcefcmooplmaimgdafohdlhacjdo)

### 線上體驗

[plaintab.kaininx.workers.dev](https://plaintab.kaininx.workers.dev)

### 本機執行

```bash
git clone https://github.com/kaininx/PlainTab.git
```

擴充模式：

1. 打開 `chrome://extensions`
2. 啟用開發者模式
3. 選擇「載入未封裝的擴充功能」
4. 選擇 PlainTab 專案目錄

網頁模式：

直接用瀏覽器打開 `index.html`。

不需要安裝依賴，也不需要建置。

## 專案結構

```text
PlainTab/
├── index.html              # 新分頁和網頁入口
├── manifest.json           # 瀏覽器擴充功能設定
├── css/                    # 頁面樣式
├── js/                     # 頁面邏輯
├── js/wallpaper/           # 桌布相關功能
├── _locales/               # 擴充功能多語言
├── docs/                   # 文件
├── icon/                   # 圖示
└── imgs/                   # 截圖和商店素材
```
