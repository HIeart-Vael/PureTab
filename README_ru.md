## ✨ Почему PlainTab?

- 🔒 **Абсолютно чистая приватность**  
  Никакие личные данные не собираются, никакие чувствительные разрешения не запрашиваются. Все обои хранятся только на вашем устройстве. История просмотров принадлежит только вам.

- 🚀 **Единая точка начала работы за одну минуту**  
  Установите `plaintab.netlify.app` в качестве домашней страницы браузера, затем установите расширение для новой вкладки. Домашняя страница, стартовая страница и новая вкладка — все в едином минималистичном стиле. Расширение **не изменяет принудительно настройки домашней страницы**, вы можете свободно использовать любой URL.

- 🧩 **Настолько лёгкое, что вы его не замечаете**  
  Никаких внешних зависимостей, чистый ванильный JavaScript. Мгновенный запуск, никаких задержек.

- 🌍 **Работает сразу, но понимает вас**  
  Автоматически определяет язык браузера (16 языков). Поддерживает Google / Bing / Baidu / DuckDuckGo. Строка поиска настраивается: «При наведении», «Всегда» или «Никогда».

## 🚀 Два способа использования

| Способ | Описание | Для кого |
|--------|----------|----------|
| 🌐 **Онлайн-страница** | Откройте [plaintab.netlify.app](https://plaintab.netlify.app) и установите как домашнюю страницу | Для тех, кто хочет просто чистую домашнюю страницу без расширений |
| 🧩 **Расширение** | Установите расширение вручную (скоро в магазинах Chrome/Edge). Новые вкладки автоматически становятся PlainTab | Для тех, кто хочет единый стиль на каждой новой вкладке |

> 💡 Рекомендуем использовать оба: домашняя страница — онлайн, новые вкладки — расширение.

### Расширение · Ручная установка (режим разработчика)
1. Клонируйте репозиторий  
   `git clone https://github.com/kaininx/PlainTab.git`
2. Откройте Chrome или Edge, перейдите в `chrome://extensions`
3. Включите **режим разработчика** в правом верхнем углу
4. Нажмите **Загрузить распакованное расширение** и выберите папку проекта
5. Готово! Откройте новую вкладку, чтобы увидеть PlainTab

## 🛠️ Как использовать

| Действие | Результат |
|----------|-----------|
| Навести курсор на правый верхний угол | Показать значки языка и настроек |
| Навести курсор на центр страницы | Строка поиска появляется (в режиме «При наведении») |
| Нажать на шестерёнку | Открыть панель обоев и дополнительных настроек |
| Нажать на значок глобуса | Переключить язык интерфейса |
| Нажать на значок поисковика | Циклически переключить Google → Bing → Baidu → DuckDuckGo |
| Нажать `Enter` в строке поиска | Поиск через выбранную систему |
| Нажать `Esc` | Закрыть все панели |

### Обои
Хранение обоев намеренно ограничено, **кеш не разрастается**:

- **Ежедневные обои Bing**: автоматически загружаются раз в день, кешируется **только одно изображение за текущий день**. Старый кеш автоматически заменяется — офлайн-доступ к вчерашнему изображению сохраняется, но хранилище не засоряется.
- **Локальные обои**: вы можете загрузить своё изображение (хранится в IndexedDB). Система сохраняет **только последнее** загруженное изображение. Одно нажатие «Сбросить на Bing дня» удаляет локальное изображение и возвращает ежедневные обновления.

### Дополнительные настройки
| Настройка | Описание |
|-----------|----------|
| Режим строки поиска | При наведении / Всегда / Никогда |
| Прозрачность значка | 0 – 1 (по умолчанию 0.45) |
| Поисковая система по умолчанию | Google / Bing / Baidu / DuckDuckGo |

> Все настройки сохраняются в `localStorage`. Без аккаунта, без облачной синхронизации.

## 🤝 Участие в разработке

Приветствуются любые Issue и Pull Request.  
Но перед началом, пожалуйста, помните о нашей общей цели:

- **Стиль PlainTab — «минимализм»**. Каждая строка кода и каждая функция тщательно взвешиваются. Если какие-то, казалось бы, отличные предложения не принимаются — это не отрицание их ценности, а стремление сохранить эту сдержанность.
- Следуйте существующему стилю кода — ванильный JS, без шагов сборки, без сторонних зависимостей.

Основной процесс:
1. Форкните репозиторий  
2. Создайте новую ветку (`feat/ваша-функция`)  
3. Зафиксируйте изменения  
4. Отправьте в ветку  
5. Создайте Pull Request
