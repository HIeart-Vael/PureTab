# 花了一百多个小时，我把浏览器新标签页重做了一遍。

起因很简单——我是个纯屏党，实在受不了现在的浏览器标签页，丑也就算了，什么时间天气新闻，对我这种打工牛马来说，每天打开新标签页都是为了打开别的网页，这种功能约等于无用，更受不了的是这个奇丑无比的默认页我每天至少要看 10 次以上！既然受不了，那必须终结它！其实早就想做了，一直想做但苦于没时间，现在下岗了，大把时间必须安排它（笑），顺便也想试试大火的 vibe coding 大法。于是就诞生了今天的主角——PlainTab · 极简新标签页。

我暂时还没在市面上找到类似的，大家都在默契的把新标签页往瑞士军刀发展，什么天气时钟磁贴便签问候语，我真不理解标签页整那么复杂干嘛，真有人看嘛，以上全部就是起因，下面来给兄弟们交答卷，前前后后烧了两亿多 token：

<img src="imgs/chrome_05.jpg" width="45%" />
PlainTab 界面支持 16 国语言（感谢 AI 让我精通 16 国语言，真的牛逼），会根据浏览器语言自动适配。界面元素全部支持隐藏，也就是说，只要你想，完全可以让新标签页打开只显示一张壁纸，别的什么都没有。界面默认显示 Bing 每日壁纸，每天自动更新。你也可以上传自己喜欢的壁纸，不限制大小（重点），最多支持 12 张，每次新开标签页会自动轮换。虽然是上传，实际存在你自己的浏览器缓存里——插件不允许访问本地文件，我也不会收集，所以你可以上传任何图片，任何（斜眼笑）。

那么代价是什么呢？答案是没有代价。我认为它解决了我的痛点，也希望能解决更多人的，互联网的精神就是分享嘛。也希望大家多多使用、喜欢的话给个好评。能解决有此需求的人的问题，自己做的东西有人用，也是一种无上的满足感。

速度上我做了极致优化。这几天几乎都在死磕速度————想同时实现不压缩、不限大小、加载快这个几乎不可能三角，不知道掉了多少头发。但现在我可以很自豪地告诉大家：全支持！项目完全是 HTML + CSS + JS 三件套原生实现，零第三方依赖，零构建步骤，无限接近浏览器原生响应。

同时我还部署了网页版，支持搜索引擎切换。托管在 Cloudflare Pages 上（没花钱的就是最好的），不过 Cloudflare 国内访问偶尔抽风。我还部署了一份到 Netlify，这两天频繁构建把积分用超了，等 5.27 恢复后大家可以试试，个人测试 Netlify 国内访问还是挺稳定的。

推荐大家网页版设为起始页（默认 Bing 壁纸，每天一张不重样），再安装 PlainTab 扩展上传自己的壁纸。那么多好看的壁纸别再只有一张当桌面，剩下的放文件夹吃灰啦！
