## ✨ PlainTab을 선택하는 이유

- 🔒 **완벽하게 깨끗한 프라이버시**  
  개인 정보를 전혀 수집하지 않으며, 민감한 권한도 요청하지 않습니다. 모든 배경화면은 사용자의 기기에만 저장됩니다. 브라우징 습관은 완전히 당신만의 것입니다.

- 🚀 **1분 만에 통일된 브라우징 시작점 완성**  
  `plaintab.netlify.app`을 브라우저 홈페이지로 설정하고, 확장 프로그램을 설치해 새 탭을 대체하세요. 홈페이지, 스타트 페이지, 새 탭이 모두 동일한 미니멀 스타일로 통일됩니다. 확장 프로그램이 **홈페이지 설정을 강제로 변경하지 않으므로**, 원하는 시작 페이지 URL을 자유롭게 사용할 수 있습니다.

- 🧩 **느껴지지 않을 정도로 가벼운**  
  외부 의존성 제로, 순수 바닐라 JavaScript. 즉시 시작되며 어떤 클릭도 느려지지 않습니다.

- 🌍 **바로 사용 가능, 하지만 당신을 이해합니다**  
  브라우저 언어 자동 감지 (16개 언어). Google / Bing / Baidu / DuckDuckGo 4가지 검색 엔진 지원. 검색창은 "호버 시", "항상 표시", "숨기기" 중 선택 가능합니다.

## 🚀 두 가지 이용 방법

| 방법 | 설명 | 추천 대상 |
|------|------|-----------|
| 🌐 **온라인 스타트 페이지** | [plaintab.netlify.app](https://plaintab.netlify.app)에 접속해 브라우저 설정에서 홈페이지/시작 페이지로 지정 | 확장 프로그램 설치 없이 깔끔한 홈페이지만 원하는 분 |
| 🧩 **브라우저 확장 프로그램** | 확장 프로그램 수동 설치 (곧 Edge / Chrome 스토어 출시). 새 탭이 자동으로 PlainTab으로 변경됩니다 | 모든 새 탭에서 동일한 미니멀 경험을 원하는 분 |

> 💡 두 가지를 함께 사용하는 것을 추천합니다: 홈페이지는 온라인 버전, 새 탭은 확장 프로그램으로.

### 브라우저 확장 프로그램 · 수동 설치 (개발자 모드)
1. 저장소 클론  
   `git clone https://github.com/kaininx/PlainTab.git`
2. Chrome 또는 Edge에서 `chrome://extensions` 열기
3. 오른쪽 상단의 **개발자 모드** 활성화
4. **압축 해제된 확장 프로그램 로드**를 클릭하고 프로젝트 폴더 선택
5. 완료! 새 탭을 열면 PlainTab이 표시됩니다

## 🛠️ 사용 방법

| 동작 | 효과 |
|------|------|
| 마우스를 오른쪽 상단으로 이동 | 언어 / 설정 아이콘 표시 |
| 마우스를 페이지 중앙 부근으로 이동 | 검색창 페이드 인 ("호버 시" 모드) |
| 톱니바퀴 아이콘 클릭 | 배경화면 및 고급 옵션 패널 열기 |
| 지구본 아이콘 클릭 | 인터페이스 언어 전환 |
| 검색 엔진 아이콘 클릭 | Google → Bing → Baidu → DuckDuckGo 순환 전환 |
| 검색창에서 `Enter` | 현재 검색 엔진으로 검색 |
| `Esc` | 모든 패널 닫기 |

### 배경화면
PlainTab의 배경화면 저장은 의도적으로 억제되어 있어 **캐시가 팽창하지 않습니다**:

- **Bing 매일 배경화면**: 하루에 한 번 자동으로 가져와 **해당 날짜의 배경화면 1장만** 캐시합니다. 오래된 캐시는 자동으로 교체되어, 다음 날 오프라인에서도 어제 이미지를 볼 수 있으면서 저장 공간 낭비를 방지합니다.
- **로컬 배경화면**: 자신의 이미지를 업로드할 수 있습니다 (IndexedDB에 저장). 시스템은 항상 **마지막으로 업로드된 1장**만 유지하며, 새 업로드는 기존 것을 대체합니다. "Bing 매일 배경화면으로 복원" 한 번의 클릭으로 로컬 이미지를 삭제하고 매일 자동 업데이트로 돌아갑니다.

### 고급 옵션
| 옵션 | 설명 |
|------|------|
| 검색창 표시 모드 | 호버 시 / 항상 표시 / 숨기기 |
| 아이콘 불투명도 | 0 – 1 (기본값 0.45) |
| 기본 검색 엔진 | Google / Bing / Baidu / DuckDuckGo |

> 모든 설정은 `localStorage`에 자동 저장됩니다. 계정 불필요, 클라우드 동기화 없음.

## 🤝 기여하기

모든 Issue와 Pull Request를 환영합니다.  
하지만 시작하기 전에, 우리의 공통된 목표를 이해해 주세요:

- **PlainTab의 스타일은 "미니멀"입니다**. 모든 코드 한 줄, 모든 기능은 비대해지지 않도록 신중하게 검토됩니다. 겉보기에 훌륭해 보이는 제안이 최종적으로 채택되지 않더라도, 그것은 그 가치를 부정하는 것이 아니라 이 절제의 원칙을 지키기 위함입니다.
- 기존 코딩 스타일을 따라주세요 — 바닐라 JS, 빌드 단계 없음, 서드파티 의존성 없음.

기본 워크플로우:
1. 저장소 포크  
2. 새 브랜치 생성 (`feat/your-feature`)  
3. 변경 사항 커밋  
4. 브랜치에 푸시  
5. Pull Request 생성

## 🙏 감사의 말

- Bing 배경화면 API는 [bing.img.run](https://bing.img.run)과 [bing.biturl.top](https://bing.biturl.top)이 제공합니다. 안정적인 서비스에 감사드립니다.
- 스크린샷에 사용된 일부 배경화면은 인터넷에서 가져왔습니다. **모든 재능 있는 창작자분들께 감사드립니다**. 본 프로젝트는 완전한 비영리 개인 프로젝트로, 모든 출처를 추적할 수 없었습니다. 귀하가 작품의 저작권자이며 전시가 권리를 침해한다고 생각되시면, 반드시 연락 주시기 바랍니다 — 즉시 사용 허가를 협의하거나 해당 이미지를 교체하겠습니다. 모든 창작자의 권리를 깊이 존중합니다.
