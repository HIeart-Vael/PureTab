Nhật ký thay đổi

• v3.1.0
  • Thư viện hình nền cục bộ: tải lên, xóa, xoay vòng nhiều hình nền với chọn hàng loạt; tái cấu trúc mã (đặt tên ngữ nghĩa, chia nhỏ hàm, chú thích thiết kế); cải thiện kiểu tiêu đề bảng; nâng cao bản địa hóa đa ngôn ngữ; sửa CSP để xóa miền JSON API.

• v3.0.5
  • Nâng cấp proxy Bing API (hỗ trợ khu vực + loại bỏ trùng lặp hình ảnh); tối ưu hóa bộ nhớ đệm hình nền (kiểm tra IDB trước khi tải xuống); thống nhất khóa localStorage; thêm tệp changelog 16 ngôn ngữ; cải thiện ghi nhật ký và ngữ nghĩa mã.

• v3.0.4
  • Đã sửa lỗi tải hình nền: danh sách cho phép CSP `img-src` thiếu nguồn Bing CDN (`www.bing.com`, `cn.bing.com`), do đó hình ảnh được chuyển hướng từ API proxy bị trình duyệt chặn.

• v3.0.3
  • Đã bỏ qua CSSOM trên đường dẫn tải trước quan trọng để hiển thị khung hình đầu tiên nhanh hơn; tăng cường CSP; xóa code thừa; thêm biểu tượng SVG và tài liệu quảng bá.

• v3.0.2
  • Hình nền thích ứng chế độ tối, tái cấu trúc code, thanh chân trang đa ngôn ngữ.

• v3.0.1
  • Đã chuyển sang Chrome Search API để tuân thủ chính sách mục đích duy nhất của CWS.

• v3.0.0
  • Viết lại toàn bộ: kiến trúc hai lớp để hiển thị hình nền không nhấp nháy với nền dự phòng. (Xem V3_NOTE.md)

• v2.2.7
  • Đã thêm README 5 ngôn ngữ, đặt chế độ tìm kiếm mặc định thành "luôn hiển thị".

• v2.2.6
  • Đã tối ưu hóa tải hình nền và bộ nhớ đệm hình thu nhỏ; dọn dẹp hằng số Bing API.

• v2.2.5
  • Đã thêm tải trước hình nền và tối ưu hóa khung hình đầu tiên để loại bỏ hiện tượng nhấp nháy trắng khi mở tab mới.

• v2.2.4
  • Đã chuyển bộ nhớ hình nền từ localStorage sang IndexedDB.

• v2.2.3
  • Đã thêm tải lên hình nền cục bộ và cải thiện trải nghiệm chuyển đổi công cụ tìm kiếm.

• v1.2.0
  • Phát hành lần đầu: phát triển từ một trang HTML đơn lẻ thành một tiện ích đầy đủ với hệ thống hình nền, giao diện 16 ngôn ngữ, bản địa hóa Chrome i18n và giấy phép MIT.
