परिवर्तन लॉग

• v3.1.0
  • स्थानीय वॉलपेपर गैलरी: एकाधिक वॉलपेपर अपलोड/हटाएं/घुमाएं, बैच फ़ाइल चयन; कोड रिफैक्टरिंग (अर्थपूर्ण नामकरण, लंबे फ़ंक्शन विभाजित, डिज़ाइन टिप्पणियाँ); पैनल शीर्षक शैलियों में सुधार; बहुभाषी स्थानीयकरण में वृद्धि; JSON API डोमेन हटाने के लिए CSP ठीक किया गया।

• v3.0.5
  • Bing API प्रॉक्सी अपग्रेड (क्षेत्रीय समर्थन + छवि डीडुप्लीकेशन); वॉलपेपर कैशिंग अनुकूलित (डाउनलोड से पहले IDB जांचें); localStorage कुंजियाँ एकीकृत; 16-भाषा changelog फ़ाइलें जोड़ी गईं; लॉगिंग और कोड अर्थविज्ञान में सुधार।

• v3.0.4
  • वॉलपेपर लोडिंग ठीक की गई: CSP `img-src` अनुमति सूची में Bing CDN origins (`www.bing.com`, `cn.bing.com`) गायब थे, जिससे प्रॉक्सी API से रीडायरेक्ट की गई छवियां ब्राउज़र द्वारा ब्लॉक हो रही थीं।

• v3.0.3
  • तेज़ पहले पेंट के लिए क्रिटिकल प्रीलोड पथ पर CSSOM को बायपास किया गया; CSP को मजबूत किया गया; डेड कोड हटाया गया; SVG आइकन और प्रमोशन सामग्री जोड़ी गई।

• v3.0.2
  • डार्क मोड अनुकूली पृष्ठभूमि, कोड संरचना पुनर्गठित, बहुभाषी फ़ुटर बार।

• v3.0.1
  • CWS एकल-उद्देश्य अनुपालन के लिए Chrome Search API पर स्विच किया गया।

• v3.0.0
