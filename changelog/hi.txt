बदलावों का लॉग

• v3.1.1 · Bing API को Promise.any रेस मोड में बदला गया और 8 सेकंड AbortController टाइमआउट जोड़ा गया; नई प्राप्ति रणनीति को दर्शाने के लिए सभी 16 भाषाओं के README अपडेट किए गए।

• v3.1.0 · स्थानीय वॉलपेपर गैलरी: अपलोड/डिलीट/रोटेट, बैच फ़ाइल चयन; कोड रिफैक्टरिंग; पैनल स्टाइल में सुधार; बहुभाषी स्थानीयकरण विस्तार; CSP फिक्स — JSON API डोमेन हटाए गए।

• v3.0.5 · Bing API प्रॉक्सी को अपग्रेड किया गया (क्षेत्रीय अनुरोध + इमेज डिडप्लिकेशन); वॉलपेपर कैश ऑप्टिमाइज़ किया गया; localStorage कुंजी नामों को एकीकृत किया गया; 16 भाषाओं में changelog जोड़ा गया; लॉग और कोड सिमैंटिक्स में सुधार।

• v3.0.4 · वॉलपेपर लोडिंग विफलता ठीक की गई: CSP img-src व्हाइटलिस्ट में Bing CDN डोमेन की कमी थी, जिससे प्रॉक्सी API रीडायरेक्ट के बाद इमेजेज़ ब्राउज़र द्वारा ब्लॉक हो रही थीं।

• v3.0.3 · CSSOM को बायपास करके पहले फ्रेम रेंडरिंग को तेज़ किया गया, CSP सुरक्षा को मजबूत किया गया, डेड कोड हटाया गया, नए SVG आइकन और प्रमोशनल मटीरियल जोड़े गए।

• v3.0.2 · डार्क मोड में अनुकूली पृष्ठभूमि, कोड संरचना पुनर्गठन, निचली बहुभाषी जानकारी पट्टी।

• v3.0.1 · Chrome Search API को एकीकृत किया गया, Chrome Web Store की एकल-उद्देश्य अनुपालन समीक्षा पास की गई।

• v3.0.0 · पूर्ण पुनर्लेखन: दोहरी-परत आर्किटेक्चर जो नए टैब की सफेद स्क्रीन फ्लिकर को खत्म करता है, सुरक्षित ग्रे बैकग्राउंड जोड़ा गया।

• v2.2.7 · अंग्रेज़ी/जापानी/कोरियाई/रूसी/सरलीकृत चीनी में नए README; डिफ़ॉल्ट खोज मोड "हमेशा दिखाएँ" में बदला गया।

• v2.2.6 · वॉलपेपर लोडिंग लॉजिक और थंबनेल कैश का अनुकूलन, Bing API स्थिरांकों का पुनर्गठन।

• v2.2.5 · वॉलपेपर प्रीलोडिंग और पहले फ्रेम का अनुकूलन, नए टैब की सफेद स्क्रीन को खत्म करना।
