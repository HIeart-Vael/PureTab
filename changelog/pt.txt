Registro de alterações

• v3.1.1
  • Mudou a API do Bing para modo de competição com Promise.any e tempo limite de 8s com AbortController; atualizou todos os 16 READMEs para refletir a nova estratégia de busca e adicionou técnicas de latência com Promise.any/AbortController.

• v3.1.0
  • Galeria de papéis de parede locais: enviar, excluir, rotacionar vários papéis de parede com seleção em lote; refatoração de código (nomes semânticos, divisão de funções, comentários de design); melhoria nos estilos de títulos dos painéis; localização multilíngue aprimorada; CSP corrigido para remover domínios de API JSON.

• v3.0.5
  • Proxy da API Bing atualizado (suporte regional + deduplicação de imagens); cache de papel de parede otimizado (verificar IDB antes de baixar); chaves localStorage unificadas; arquivos changelog em 16 idiomas adicionados; registros e semântica de código aprimorados.

• v3.0.4
  • Corrigido o carregamento do papel de parede: a lista de permissões CSP `img-src` não incluía as origens do CDN do Bing (`www.bing.com`, `cn.bing.com`), então as imagens redirecionadas das APIs proxy eram bloqueadas pelo navegador.

• v3.0.3
  • CSSOM contornado no caminho crítico de pré-carregamento para primeira renderização mais rápida; CSP reforçado; código morto removido; ícones SVG e materiais promocionais adicionados.

• v3.0.2
  • Fundo adaptativo para modo escuro, código reestruturado, barra inferior multilíngue.

• v3.0.1
  • Migrado para Chrome Search API para conformidade com a política de propósito único da CWS.

• v3.0.0
  • Reescrita completa: arquitetura de camada dupla para renderização de papel de parede sem cintilação com fundo de reserva. (Veja V3_NOTE.md)

• v2.2.7
  • Adicionados READMEs em 5 idiomas, modo de pesquisa padrão alterado para "sempre visível".

• v2.2.6
  • Carregamento de papel de parede e cache de miniaturas otimizados; constantes da API do Bing limpas.

• v2.2.5
  • Pré-carregamento de papel de parede e otimização do primeiro quadro adicionados para eliminar o flash branco ao abrir uma nova guia.

• v2.2.4
  • Armazenamento do papel de parede migrado de localStorage para IndexedDB.

• v2.2.3
  • Upload de papel de parede local adicionado e experiência de troca de mecanismo de pesquisa melhorada.

• v1.2.0
  • Lançamento inicial: evoluiu de uma única página HTML para uma extensão completa com sistema de papel de parede, interface em 16 idiomas, localização Chrome i18n e licença MIT.
