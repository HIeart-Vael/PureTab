Registro delle modifiche

• v3.0.4
  • Risolto il caricamento dello sfondo: la whitelist CSP `img-src` non includeva le origini CDN di Bing (`www.bing.com`, `cn.bing.com`), quindi le immagini reindirizzate dalle API proxy venivano bloccate dal browser.

• v3.0.3
  • Bypassato CSSOM sul percorso critico di precaricamento per un primo rendering più veloce; rafforzato CSP; rimosso codice morto; aggiunte icone SVG e materiale promozionale.

• v3.0.2
  • Sfondo adattivo per modalità scura, codice ristrutturato, barra inferiore multilingue.

• v3.0.1
  • Passaggio a Chrome Search API per la conformità a scopo unico CWS.

• v3.0.0
  • Riscrittura completa: architettura a doppio strato per rendering dello sfondo senza sfarfallio con sfondo di riserva. (Vedi V3_NOTE.md)

• v2.2.7
  • Aggiunti README in 5 lingue, modalità di ricerca predefinita impostata su "sempre visibile".

• v2.2.6
  • Ottimizzato il caricamento dello sfondo e la cache delle miniature; pulite le costanti dell'API Bing.

• v2.2.5
  • Aggiunti precaricamento dello sfondo e ottimizzazione del primo fotogramma per eliminare il lampo bianco all'apertura di una nuova scheda.

• v2.2.4
  • Migrato l'archiviazione dello sfondo da localStorage a IndexedDB.

• v2.2.3
  • Aggiunto caricamento sfondo locale e migliorata l'esperienza di cambio motore di ricerca.

• v1.2.0
  • Rilascio iniziale: evoluto da una singola pagina HTML a un'estensione completa con sistema di sfondi, interfaccia in 16 lingue, localizzazione Chrome i18n e licenza MIT.
