Journal des modifications

• v3.0.4
  • Correction du chargement du fond d'écran : la liste blanche CSP `img-src` ne contenait pas les origines du CDN Bing (`www.bing.com`, `cn.bing.com`), donc les images redirigées depuis les API proxy étaient bloquées par le navigateur.

• v3.0.3
  • Contournement de CSSOM sur le chemin critique de préchargement pour un premier affichage plus rapide ; CSP renforcée ; code mort supprimé ; icônes SVG et supports promotionnels ajoutés.

• v3.0.2
  • Fond adaptatif pour mode sombre, code restructuré, barre inférieure multilingue.

• v3.0.1
  • Passage à Chrome Search API pour la conformité à usage unique de CWS.

• v3.0.0
  • Réécriture complète : architecture double couche pour un rendu du fond d'écran sans scintillement avec fond de secours. (Voir V3_NOTE.md)

• v2.2.7
  • Ajout de README en 5 langues, mode de recherche par défaut réglé sur « toujours visible ».

• v2.2.6
  • Optimisation du chargement du fond d'écran et du cache de miniatures ; nettoyage des constantes de l'API Bing.

• v2.2.5
  • Préchargement du fond d'écran et optimisation de la première image ajoutés pour éliminer le flash blanc à l'ouverture d'un nouvel onglet.

• v2.2.4
  • Migration du stockage du fond d'écran de localStorage vers IndexedDB.

• v2.2.3
  • Ajout du téléchargement de fond d'écran local et amélioration de l'expérience de changement de moteur de recherche.

• v1.2.0
  • Version initiale : évolution d'une simple page HTML vers une extension complète avec système de fond d'écran, interface en 16 langues, localisation Chrome i18n et licence MIT.
