Änderungsprotokoll

• v3.0.4
  • Hintergrundbild-Ladefehler behoben: CSP `img-src`-Whitelist enthielt keine Bing-CDN-Ursprünge (`www.bing.com`, `cn.bing.com`), sodass von Proxy-APIs umgeleitete Bilder vom Browser blockiert wurden.

• v3.0.3
  • CSSOM auf dem kritischen Preload-Pfad umgangen für schnelleres erstes Rendering; CSP verstärkt; toten Code entfernt; SVG-Icons und Werbematerialien hinzugefügt.

• v3.0.2
  • Dunkelmodus-adaptiver Hintergrund, Codebasis umstrukturiert, mehrsprachige Fußzeile.

• v3.0.1
  • Auf Chrome Search API umgestellt für CWS Single-Purpose-Compliance.

• v3.0.0
  • Komplette Neuentwicklung: Zwei-Schichten-Architektur für flimmerfreies Hintergrund-Rendering mit Fallback-Hintergrund. (Siehe V3_NOTE.md)

• v2.2.7
  • READMEs in 5 Sprachen hinzugefügt, Standardsuchmodus auf "Immer anzeigen" gesetzt.

• v2.2.6
  • Hintergrundbild-Ladung und Thumbnail-Caching optimiert; Bing-API-Konstanten bereinigt.

• v2.2.5
  • Vorabladen von Hintergrundbildern und First-Frame-Optimierung hinzugefügt, um den weißen Blitz beim Öffnen eines neuen Tabs zu beseitigen.

• v2.2.4
  • Hintergrundbild-Speicherung von localStorage auf IndexedDB migriert.

• v2.2.3
  • Lokales Hintergrundbild-Upload und verbesserte Suchmaschinen-Wechsel-UX hinzugefügt.

• v1.2.0
  • Erstveröffentlichung: Entwicklung von einer einzelnen HTML-Seite zu einer vollständigen Erweiterung mit Hintergrundbildsystem, 16-sprachiger Oberfläche, Chrome i18n Lokalisierung und MIT-Lizenz.
