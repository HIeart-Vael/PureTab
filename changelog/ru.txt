История изменений

• v3.0.4
  • Исправлена загрузка обоев: в белом списке CSP `img-src` отсутствовали источники CDN Bing (`www.bing.com`, `cn.bing.com`), поэтому изображения, перенаправленные через прокси-API, блокировались браузером.

• v3.0.3
  • Обход CSSOM на критическом пути предзагрузки для ускорения первой отрисовки; усилен CSP; удалён мёртвый код; добавлены SVG-иконки и промо-материалы.

• v3.0.2
  • Адаптивный фон для тёмной темы, реструктурированный код, многоязычная нижняя панель.

• v3.0.1
  • Переход на Chrome Search API для соответствия политике единого назначения CWS.

• v3.0.0
  • Полная переработка: двухслойная архитектура для отображения обоев без мерцания с резервным серым фоном. (См. V3_NOTE.md)

• v2.2.7
  • Добавлены README на 5 языках, режим поиска по умолчанию изменён на «всегда отображать».

• v2.2.6
  • Оптимизирована загрузка обоев и кэширование миниатюр; очищены константы Bing API.

• v2.2.5
  • Добавлена предзагрузка обоев и оптимизация первого кадра для устранения белой вспышки при открытии новой вкладки.

• v2.2.4
  • Хранилище обоев перенесено из localStorage в IndexedDB.

• v2.2.3
  • Добавлена загрузка локальных обоев и улучшено переключение поисковых систем.

• v1.2.0
  • Первый релиз: эволюция от одной HTML-страницы до полноценного расширения с системой обоев, 16-язычным интерфейсом, локализацией Chrome i18n и лицензией MIT.
