Dziennik zmian

• v3.1.0
  • Galeria lokalnych tapet: przesyłanie, usuwanie, rotacja wielu tapet z wyborem wsadowym; refaktoryzacja kodu (nazewnictwo semantyczne, podział funkcji, komentarze projektowe); ulepszone style tytułów paneli; rozszerzona lokalizacja wielojęzyczna; poprawione CSP w celu usunięcia domen JSON API.

• v3.0.5
  • Zaktualizowano proxy Bing API (obsługa regionalna + deduplikacja obrazów); zoptymalizowano buforowanie tapet (sprawdzanie IDB przed pobraniem); ujednolicono klucze localStorage; dodano pliki changelog w 16 językach; ulepszono rejestrowanie i semantykę kodu.

• v3.0.4
  • Naprawiono ładowanie tapety: na liście dozwolonych CSP `img-src` brakowało źródeł CDN Bing (`www.bing.com`, `cn.bing.com`), więc obrazy przekierowywane z proxy API były blokowane przez przeglądarkę.

• v3.0.3
  • Ominięto CSSOM na ścieżce krytycznej wstępnego ładowania dla szybszego pierwszego renderowania; wzmocniono CSP; usunięto martwy kod; dodano ikony SVG i materiały promocyjne.

• v3.0.2
  • Adaptacyjne tło dla trybu ciemnego, zrestrukturyzowany kod, wielojęzyczny pasek dolny.

• v3.0.1
  • Przełączono na Chrome Search API w celu zgodności z polityką jednego celu CWS.

• v3.0.0
  • Całkowita przebudowa: dwuwarstwowa architektura do renderowania tapety bez migotania z zapasowym szarym tłem. (Zobacz V3_NOTE.md)

• v2.2.7
  • Dodano README w 5 językach, domyślny tryb wyszukiwania ustawiono na "zawsze widoczny".

• v2.2.6
  • Zoptymalizowano ładowanie tapet i buforowanie miniatur; uporządkowano stałe API Bing.

• v2.2.5
  • Dodano wstępne ładowanie tapety i optymalizację pierwszej klatki, aby wyeliminować białe migotanie podczas otwierania nowej karty.

• v2.2.4
  • Przeniesiono przechowywanie tapet z localStorage do IndexedDB.

• v2.2.3
  • Dodano przesyłanie lokalnej tapety i poprawiono UX zmiany wyszukiwarki.

• v1.2.0
  • Pierwsze wydanie: ewolucja z pojedynczej strony HTML do pełnego rozszerzenia z systemem tapet, 16-języcznym interfejsem, lokalizacją Chrome i18n i licencją MIT.
