변경 로그

• v3.1.0
  • 로컬 배경화면 갤러리: 여러 배경화면 업로드/삭제/순환, 일괄 파일 선택; 코드 리팩토링(의미적 명명, 긴 함수 분할, 설계 주석); 설정 및 언어 패널 제목 스타일 개선; 다국어 현지화 텍스트 강화; CSP 수정으로 JSON API 도메인 제거.

• v3.0.5
  • Bing API 프록시 업그레이드(지역 지원 + 이미지 중복 제거); 배경화면 캐싱 로직 최적화(다운로드 전 IDB 확인); localStorage 키 통일; 16개 언어 changelog 파일 추가; 로깅 및 코드 의미 개선.

• v3.0.4
  • 배경화면 로딩 수정: CSP `img-src` 허용 목록에 Bing CDN 원본(`www.bing.com`, `cn.bing.com`)이 누락되어 프록시 API에서 리디렉션된 이미지가 브라우저에서 차단되었습니다.

• v3.0.3
  • 더 빠른 첫 페인트를 위해 중요 프리로드 경로에서 CSSOM 우회, CSP 강화, 데드 코드 제거, SVG 아이콘 및 프로모션 자료 추가.

• v3.0.2
  • 다크 모드 적응형 배경, 코드 구조 재구성, 다국어 하단 바.

• v3.0.1
  • CWS 단일 목적 정책 준수를 위해 Chrome Search API로 전환.

• v3.0.0
  • 전체 재작성: 폴백 배경과 함께 깜박임 없는 배경화면 렌더링을 위한 이중 레이어 아키텍처. (V3_NOTE.md 참조)

• v2.2.7
  • 5개 언어 README 추가, 기본 검색 모드를 "항상 표시"로 설정.

• v2.2.6
  • 배경화면 로딩 및 썸네일 캐싱 최적화, Bing API 상수 정리.

• v2.2.5
  • 새 탭을 열 때 빈 화면 깜박임을 없애기 위한 배경화면 프리로딩 및 첫 프레임 최적화 추가.

• v2.2.4
  • 배경화면 저장소를 localStorage에서 IndexedDB로 이전.

• v2.2.3
  • 로컬 배경화면 업로드 추가 및 검색 엔진 전환 UX 개선.

• v1.2.0
  • 최초 릴리스: 단일 HTML 페이지에서 배경화면 시스템, 16개 언어 UI, Chrome i18n 로케일 및 MIT 라이선스를 갖춘 완전한 확장 기능으로 발전.
