更新履歴

• v3.1.1
  • Bing API を Promise.any による競合モードに切り替え、8 秒の AbortController タイムアウトを追加；16 言語すべての README を更新し、新しい取得戦略と Promise.any/AbortController のレイテンシー最適化技術を反映。

• v3.1.0
  • ローカル壁紙ギャラリー：複数壁紙のアップロード/削除/ローテーション、一括ファイル選択；コードリファクタリング（意味的命名、長い関数の分割、設計コメント）；設定・言語パネルのタイトルスタイル改善；多言語ローカライズテキストの強化；CSP 修正で JSON API ドメインを削除。

• v3.0.5
  • Bing API プロキシのアップグレード（地域対応 + 画像重複排除）；壁紙キャッシュロジックの最適化（ダウンロード前に IDB をチェック）；localStorage キーの統一；16言語 changelog ファイルを追加；ログ情報とコードの意味を改善。

• v3.0.4
  • 壁紙読み込みを修正：CSP `img-src` 許可リストに Bing CDN オリジン（`www.bing.com`、`cn.bing.com`）が含まれておらず、プロキシ API からリダイレクトされた画像がブラウザにブロックされていました。

• v3.0.3
  • クリティカルなプリロードパスで CSSOM をバイパスして初回描画を高速化、CSP を強化、デッドコードを削除、SVG アイコンとプロモーション素材を追加。

• v3.0.2
  • ダークモード対応の背景、コード構造を再構築、多言語フッターバー。

• v3.0.1

• v3.0.0
