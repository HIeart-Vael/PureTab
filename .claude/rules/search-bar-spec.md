# 搜索栏需求规格

## 行为概述

搜索栏是新标签页的核心交互入口——用户在这里输入关键词、按回车搜索。它浮动在壁纸之上，半透明磨砂玻璃风格，默认居中显示。

### 三种显示模式

| 模式 | 行为 |
|------|------|
| 始终显示 | 搜索栏永远可见，不做任何操作也不会消失 |
| 悬停显示 | 鼠标移到屏幕中央区域时才出现，移开消失。打字时不会消失 |
| 始终隐藏 | 搜索栏永远不显示 |

默认模式为「始终显示」。

### 搜索引擎

**网页模式下**支持四种搜索引擎：

| 引擎 | 搜索地址格式 |
|------|------------|
| Google | https://www.google.com/search?q=关键词 |
| Bing | https://www.bing.com/search?q=关键词 |
| Baidu | https://www.baidu.com/s?wd=关键词 |
| DuckDuckGo | https://duckduckgo.com/?q=关键词 |

默认引擎为 Google。点击搜索栏右侧的引擎图标可循环切换到下一个引擎。图标是内嵌 SVG，无需外部图片文件。

**扩展模式下**只能使用浏览器自带的默认搜索引擎。引擎图标变成静态放大镜，不可点击切换。设置中的搜索引擎选项也完全隐藏。

### 搜索执行

1. 用户在搜索框输入关键词
2. 按回车键
3. 系统去除首尾空格
4. 关键词为空 → 不执行搜索
5. 扩展模式 → 在当前标签页用浏览器内置搜索
6. 网页模式 → 当前页面跳转到搜索引擎结果页

### 搜索栏外观

搜索框呈胶囊形，居中浮动在页面上。搜索框内左侧为输入区域（透明背景，白色文字，18px 字号），右侧为搜索引擎图标。桌面端宽度 560px，小屏幕（480px 以下）自动变宽至 90vw。

### 可自定义的视觉属性

通过设置面板（模态窗口 → 界面分页）可调整：

- **垂直位置：** 顶部 / 中上 / 居中 / 中下 / 底部，默认居中
- **圆角样式：** 胶囊(很圆) / 圆角(略方) / 直角(几乎不圆)，默认胶囊
- **壁纸遮罩：** 壁纸之上叠加一层半透明黑色，提升白色文字的阅读清晰度。不透明度可调（0~0.6），默认无遮罩

所有调整即时生效，无需刷新。

## 约束清单

### 显隐行为

- 「始终显示」模式下搜索栏永不消失
- 「悬停显示」模式在鼠标进入屏幕中央区域时出现，移出 150ms 后消失
- 「始终隐藏」模式下搜索栏完全不显示
- 搜索框聚焦时（正在打字）搜索栏绝不消失，不受鼠标位置影响
- 命令面板打开时，即使鼠标在屏幕中心，搜索栏也不出现
- 从一种模式切换到另一种时即时生效

### 搜索执行

- 关键词为空 → 不执行搜索，不跳转
- 仅去除首尾空格，中间的空格保留
- 扩展模式下搜索引擎选项不可见、强制用浏览器默认引擎
- 网页模式下搜索引擎持久化保存，下次打开新标签页恢复

### 外观

- 桌面宽度 560px，窗口不足 480px 时自动变为 90vw
- 搜索栏所有视觉属性通过设置面板调整，修改即时生效
- 圆角最大 28px（胶囊），最小 4px（直角）
- 壁纸遮罩覆盖全屏，不响应鼠标点击（不影响下层交互）

### 图标

- 网页模式下点击引擎图标 → 循环切换（Google→Bing→Baidu→DuckDuckGo→Google）
- 图标悬停有放大效果
- 扩展模式下图标为静态放大镜，不可点击

### 语言适配

- 搜索框占位符文字随界面语言变化（如中文显示「在 Web 中搜索…」）
- 引擎图标的悬停提示也随语言变化
