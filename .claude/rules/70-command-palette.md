# 命令面板需求规格

## 行为概述

命令面板是新标签页的快捷链接管理中心。用户可以通过快捷键打开面板，在面板中搜索、添加、编辑、删除和管理各类快捷链接。

### 两套独立面板

PlainTab 提供两套完全独立的命令面板，各自管理不同的快捷链接集合：

| 面板 | 默认快捷键 | 鼠标打开 |
|------|-----------|---------|
| 普通面板 | Ctrl+K | 双击页面空白区域 |
| 隐藏面板 | Ctrl+Shift+K | 中键点击 |

两套面板共用同一套界面，但各自独立运行——每套面板只显示自己的快捷链接，添加、编辑、删除等操作只影响自己范围内的数据。快捷键可在模态窗口的「快捷键」分页中自定义。

互斥规则：普通面板打开时按隐藏面板的快捷键不切换，面板内提示「当前普通模式 — 请按 Esc 关闭后再试」；反之亦然。提示 2 秒后自动消失。

### 面板界面

命令面板从屏幕上方滑入，半透明深色背景带磨砂效果。顶部一行是**命令栏**（固定不换行），列出所有可用命令按钮：

普通面板命令栏：`add  edit  delete  hide  recent  import  export  reset  clear  help  ☰/⊞`

隐藏面板命令栏：`add  edit  delete  unhide  recent  import  export  reset  clear  help  ☰/⊞`

末尾的 ☰/⊞ 按钮切换列表/图标两种视图模式。

命令栏下方是搜索框，用于过滤快捷链接（匹配名称或网址，不区分大小写）。再下方是内容区——显示快捷链接列表或当前操作的表单。

### 快捷链接归属

一条快捷链接只能属于普通面板或隐藏面板之一，不能同时属于两边：
- 普通面板显示：所有快捷链接中未被标记为隐藏的
- 隐藏面板显示：所有快捷链接中被标记为隐藏的

### 命令行为

**add — 添加快捷链接：**
显示表单：名称（可留空）+ 网址。网址输入框旁有下载图标按钮，点击后系统尝试从目标网页抓取标题自动填入名称。网址必须为 http:// 或 https:// 协议（缺少协议时自动补 https://），拒绝危险协议。新链接的网址与现有全部链接比对（不区分 Normal/Hidden，统一去重），重复则报错。扩展模式首次使用时可能弹出权限请求。添加成功后短暂显示确认动画（圆形图标 + 网站图标 + "已添加"），2 秒后回到主列表。若当前在隐藏模式下添加，新链接自动标记为隐藏。

**edit — 编辑快捷链接：**
显示当前面板范围内的快捷链接列表，点击某一项进入编辑表单（预填当前名称和网址），提交后更新链接并刷新图标。编辑不改变该链接的 Normal/Hidden 归属。

**delete — 删除快捷链接：**
显示当前面板范围内的快捷链接列表，每项右上角有红色删除按钮始终可见。点击后从系统中彻底移除（包括图标数据和最近访问记录）。支持列表和图标两种视图模式的删除显示。

**hide / unhide — 隐藏 / 取消隐藏：**
仅在对应的面板中显示（普通面板显示 hide，隐藏面板显示 unhide）。点击某项后切换该链接的隐藏状态，即时刷新列表。

**recent — 最近访问：**
显示当前面板范围内的最近访问过的快捷链接（最多 10 条）。

**import — 导入书签：**
弹出文件选择器，接受浏览器导出的书签文件（.html / .htm）。自动解析其中的链接，全量网址去重后批量添加。

**export — 导出：**
将所有快捷链接导出为 JSON 文件下载，文件名包含日期。

**reset — 重置计数：**
将当前面板范围内所有链接的使用频率计数归零。

**clear — 清空：**
面板内二次确认后，清空全部快捷链接、图标数据、最近访问、隐藏标记。确认后返回主列表。

**help — 帮助：**
列表显示所有命令及其功能说明。

### 视图模式

列表模式：单列显示，每项包含圆形图标 + 名称 + 网址（网址超长截断显示）。

图标模式：推荐区（如果开启）+ A-Z 区双网格显示。每页 15 项（有推荐区时推荐占 5 格 + A-Z 区 10 格），底部分页圆点，支持滚轮翻页和左右滑入动画。每格显示大圆形图标 + 名称标签。

长名称自动启用跑马灯滚动效果。

推荐区：开启推荐且无搜索词时显示，按使用频率取前 5 名（仅限当前面板范围内）。出现在推荐区的不会同时出现在 A-Z 区。

A-Z 区：排除推荐项后按名称字母升序排列，分页显示。

搜索：全量过滤当前面板范围内匹配名称或网址的链接，搜索结果统一 15 项/页。

### 图标

每个快捷链接有一个圆形图标：
- 若该网站的 favicon 已成功获取 → 显示网站真实图标
- 若 favicon 获取失败或为默认占位图 → 显示彩色同心圆 + 首字母

图标来源为 DuckDuckGo 的图标服务，全球可用。新增链接时先显示首字母，后台异步加载真实图标，加载成功后自动替换。系统会检测图标是否为服务商的默认占位图，是的话回退到首字母显示。

### 导航

| 当前状态 | Escape 行为 |
|---------|------------|
| 主列表 | 关闭面板 |
| 表单 / 编辑 / 删除 / 最近 / 帮助 | 返回主列表 |
| 反馈动画中 | 不响应（2 秒自动返回） |

- 主列表和最近使用模式下，点击链接 → 当前页面跳转
- 编辑 / 删除 / 隐藏模式下，点击不跳转
- 点击面板外遮罩 → 关闭面板

### 动效

打开：遮罩渐显 + 面板从上方滑入，约 0.2 秒
关闭：遮罩渐隐，约 0.15 秒

## 约束清单

### 数据

- 快捷链接数量无上限
- 最近访问最多保存 10 条
- 每条快捷链接存储：名称、网址、使用频率、添加时间
- 网址去重：大小写不敏感，不区分 Normal/Hidden，全量比对
- 一条链接不能同时属于 Normal 和 Hidden

### URL 安全

- 只允许 http:// 和 https:// 协议
- 缺少协议自动补 https://
- 拒绝 javascript:、file: 等危险协议
- 格式必须为「协议://主机名/路径」

### 添加功能

- 名称留空 → 提交时自动用域名作为名称
- 抓取网页标题功能需要跨域权限，扩展模式下首次使用时弹出授权对话框
- 抓取失败或超时 → 回退使用域名
- 图标加载超时（3 秒）→ 显示首字母

### 导入

- 接受 .html / .htm 文件
- 从书签文件中正则提取链接和标题
- 全量 URL 去重后批量添加
- 不关闭面板

### 导出

- 输出 JSON 格式
- 文件名格式：`plaintab-shortcuts-年-月-日.json`
- 不关闭面板

### 视图

- 视图模式持久化保存，全局生效（主列表、编辑、删除、隐藏/取消隐藏、最近）
- 图标模式下推荐区不参与翻页
- 有推荐时推荐区和 A-Z 区之间有额外间距
- 图标模式下网格空位显示空白格占位

### 面板互斥

- 一个面板打开时按另一个的快捷键 → 不切换，提示用户先关当前面板
- 提示 2 秒后自动消失
- 关闭面板时自动重置为普通模式（下次打开默认是普通面板）
