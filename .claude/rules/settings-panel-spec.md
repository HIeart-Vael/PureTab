# 设置面板需求规格

## 一、概述

设置面板是用户配置 PlainTab 行为的入口。它从右上角齿轮按钮打开，是一个磨砂玻璃风格的浮动面板。面板里包含：壁纸信息、本地壁纸上传/重置按钮、高级选项（搜索栏、透明度、搜索引擎、命令面板快捷键、推荐开关）。

注意：本地壁纸的图库管理（上传后的浏览、删除、拖拽排序）逻辑上属于「本地图库」模块的需求，但图库 UI 渲染在设置面板内部。

---

## 二、打开与关闭

### 打开

- 点击右上角齿轮按钮（`#settingsBtn`）→ 打开
- 快捷键 `Ctrl+Shift+W` → 打开（如果没有则关闭）
- 如果语言面板开着 → 先关语言面板，再打开设置
- 如果命令面板开着 → 不会同时操作（全局交互保证互斥）
- 已经开着 → 不重复操作（幂等）
- 打开时齿轮按钮旋转 60 度

### 关闭

- 按 Escape → 关闭
- 点击面板外的任何地方 → 关闭
- 打开语言面板 → 自动关闭
- 已经关了 → 不重复操作

### 面板位置和样式

- 位置：右上角，齿轮按钮正下方（top: 78px, right: 22px）
- 磨砂玻璃背景、半透明边框、深色投影
- 打开/关闭有缩放+位移动画（0.3 秒）
- 鼠标在面板上时角落按钮保持可见（不自动隐藏）

## 三、面板内容结构

```
设置面板
├─ 标题：「🖼️  壁纸」
├─ 壁纸信息行（显示当前壁纸来源）
├─ [选择本地壁纸] 按钮
├─ [恢复 Bing 每日壁纸] 按钮
├─ 高级选项区（折叠）
│   ├─ [高级选项] 按钮（点击展开/折叠）
│   ├─ GitHub 链接图标
│   └─ 展开后内容：
│       ├─ 搜索栏显示（下拉框：悬停/始终/隐藏）
│       ├─ 图标不透明度（滑块 0~1 + 数字输入框 联动）
│       ├─ 搜索引擎（下拉框：Google/Bing/Baidu/DDG）
│       ├─ 命令面板快捷键（文本输入框）
│       ├─ 显示推荐（复选框）
│       └─ [恢复默认设置] 按钮
```

## 四、各控件的需求

### 4.1 壁纸信息行

- 显示一行文字说明当前壁纸来源
- Bing 模式：显示「🌄 Bing 每日壁纸」（多语言）
- 本地模式：显示「📷 本地壁纸」（多语言）
- 壁纸切换后文字自动更新

### 4.2 「选择本地壁纸」按钮

- 点击后弹出系统文件选择器
- 只接受图片文件
- 可以一次选多张（按住 Ctrl/Shift 多选）
- 选中后自动上传，上限 12 张（已存满则只能上传剩余空位数量）
- 第一张上传成功的设为当前壁纸，其余的静默保存
- 更多上传细节见「本地图库」需求

### 4.3 「恢复 Bing 每日壁纸」按钮

- 点击后弹出确认对话框（用浏览器原生 `confirm`）
- 文字：「确定删除所有本地壁纸？此操作不可撤销。」（多语言）
- 用户确认 → 删除所有本地图片、清理相关存储、切换回 Bing 模式
- 用户取消 → 什么都不做
- 如果当前就是 Bing 模式，此按钮以次要样式显示

### 4.4 高级选项区

- 默认折叠，点击「高级选项」按钮展开
- 展开/折叠有动画过渡
- 展开后显示所有高级设置的控件

### 4.5 搜索栏显示模式

- 下拉框，三个选项：悬停时显示 / 始终显示 / 始终隐藏
- 切换后立即对搜索栏生效
- 默认值：始终显示

### 4.6 图标不透明度

- 滑块（range 0~1，步长 0.01）+ 数字输入框联动
- 拖滑块 → 数字输入框同步更新，反之亦然
- 实时生效——拖动的瞬间图标透明度就变
- 控制的目标：右上角齿轮按钮、语言按钮、搜索引擎图标
- 通过 CSS 变量 `--icon-opacity` 统一控制，改一处全部生效
- 默认值：0.45

### 4.7 搜索引擎

- 下拉框，四个选项：Google / Bing / Baidu / DuckDuckGo
- 切换后立即改变搜索引擎图标和搜索行为
- 默认值：Google
- **扩展模式下这一整行隐藏**（Chrome 商店政策要求只能用浏览器默认引擎）

### 4.8 命令面板快捷键

- 文本输入框，显示当前快捷键（默认 `Ctrl+K`）
- 用户可以直接在框里按键录制新快捷键
- 保存到 `ptab_shortcut_hotkey`
- 这只是 Normal 面板的快捷键，Hidden 面板的快捷键在命令面板内部管理

### 4.9 显示推荐

- 复选框，打勾/不打勾
- 控制命令面板里是否显示「推荐」区（TOP 5 高频快捷链接）
- 默认打勾（显示推荐）

### 4.10 「恢复默认设置」按钮

- 点击后把以下设置恢复出厂值：
  - 搜索栏显示模式 → 始终显示
  - 图标不透明度 → 0.45
  - 搜索引擎 → Google
  - 命令面板快捷键 → Ctrl+K
  - 显示推荐 → 勾上
- 恢复后控件 UI 同步刷新
- 不涉及壁纸数据、语言设置、快捷链接数据

## 五、面板互斥

设置面板和语言面板不能同时打开：

- 打开设置面板 → 自动关闭语言面板
- 打开语言面板 → 自动关闭设置面板
- 打开命令面板 → 设置面板不受影响（命令面板是覆盖在最上层的全屏遮罩）

---

## 六、扩展模式的特殊处理

在 Chrome 扩展版本中：
- 「搜索引擎」那一行隐藏
- 搜索引擎图标变成静态的放大镜（不可点击切换）
- 设置面板的其他控件保持不变

---

## 七、各种情况处理

| 情况 | 处理 |
|------|------|
| 点击面板外空白 | 关闭面板 |
| 面板开着时按 Escape | 关闭面板 |
| 多次快速点击齿轮按钮 | 幂等——不会重复打开 |
| 面板开着时刷新页面 | 面板关闭（页面重载） |
| 本地模式下图库加载失败 | 图库区域显示为空，不影响其他控件 |
| localStorage 写入失败 | 设置仍在当前会话生效，下次打开可能丢失 |
| 透明度被设为极端值（0 或 1） | 允许——用户可能想要完全隐藏或完全不透明 |

## 八、与其它模块的关系

| 依赖谁 | 依赖什么 |
|--------|---------|
| 壁纸系统 | 壁纸信息显示、模式切换（上传/重置） |
| 本地图库 | 本地上传后图库刷新、删除后图库刷新 |
| 搜索栏 | 搜索模式、搜索引擎、透明度的设置控件联动 |
| 命令面板 | 快捷键设置、推荐开关 |
| 语言系统 | 面板所有文字多语言 |
| 数据存储 | 所有设置项的持久化 |

| 被谁依赖 | 怎么依赖 |
|---------|---------|
| 全局交互 | 齿轮按钮触发打开/关闭、Escape 关闭、点击面板外关闭、鼠标在面板上阻止角落按钮隐藏 |
| 本地图库 | 图库 UI 在设置面板内渲染，面板打开时刷新图库 |
