# 设置面板需求规格

## 行为概述

设置系统分为两层：一级面板和模态窗口。

**一级面板**是快速信息面板——点击右上角齿轮按钮弹出，显示当前壁纸来源、预览缩略图、以及跳到完整设置的入口。日常使用只需一级面板。

**模态窗口**是完整设置中心——通过快捷键（默认 Ctrl+Shift+W）或一级面板的「高级设置」按钮打开，居中浮动显示，包含四个分页：界面、壁纸、快捷键、关于。

### 一级面板

位于右上角齿轮按钮正下方，是一个带磨砂玻璃效果的小面板。内容从上到下依次是：

**壁纸来源标签：** 彩色小标签标明当前壁纸来源。Bing 显示蓝色、本地上传显示绿色、文件夹显示黄色、RSS 显示橙色、API 显示紫色。

**壁纸计数：** 对于有限额的来源（本地上传、RSS）显示「已用/总额」格式，如「3/12」；文件夹显示「N 张」；Bing 和 API 不显示计数。

**副信息行：** 灰色小字显示当前壁纸的补充信息——Bing 显示市场区域和缓存日期，本地上传显示「本地轮换」，文件夹显示路径，RSS 显示 Feed 地址，API 显示端点域名。

**画廊预览区：** 显示当前壁纸来源的缩略图网格。Bing 和 API 来源显示单张 16:9 当前壁纸缩略图。本地上传和文件夹模式下，画廊功能完整展开：

**网格布局（本地上传/文件夹模式）：** 所有格子统一 16:9 比例。列数和大小随图片数量自适应——1 张铺满整行、2 张并排、3 张三列、4 张 2×2、5~12 张三列流式。不足一行自然留空。悬停时格子略微放大并出现阴影。

**缩略图：** 优先使用已缓存的压缩缩略图，若缺失则从原图即时生成。原图也丢失时该格子显示为空，不影响其他。

**删除按钮：** 每张缩略图右上角红色删除按钮，悬停时出现。单张图片时也有。点击删除后，若删光了自动切回 Bing。删除流程保证崩溃安全——先断引用，再删缩略图和元数据，最后删原图。

**「＋」上传按钮：** 不满 12 张时末尾有全宽长条按钮。点击弹出文件选择器，支持多选。上传去重（同名同大小跳过）、批量串行处理，超出空位自动跳过。第一张上传成功后立即展示为当前壁纸。

**拖拽排序：** 2 张及以上时可用。长按约 0.3 秒触发，被拖拽卡片浮起跟随指针，原位出现虚线占位框。指针下的格子出现高亮边框提示落点位置。松手后卡片滑入新位置，顺序即时保存。松手在外部则回原位不保存。

**Blob URL 生命周期：** 画廊显示期间若缩略图缺失，会从原图创建临时资源用于显示。面板关闭时必须释放这些资源，防止内存泄漏。

**底部栏：** 左侧「高级设置」按钮打开模态窗口，右侧 GitHub 图标链接，底部显示 PlainTab 版本号。

一级面板与语言面板互斥——打开其中一个会自动关闭另一个。关闭方式：点击面板外区域、按 Escape、或再次点击齿轮按钮。

### 模态窗口

居中浮动的设置窗口，左侧竖排四个分页标签（界面、壁纸、快捷键、关于），右侧为对应内容区。打开时有关闭动画（从上方轻微滑入）。关闭方式：按 Escape 或点击窗口外的半透明遮罩区域。

#### 界面分页

所有与视觉外观相关的设置集中在此：

| 设置项 | 可选值 | 默认值 | 说明 |
|--------|--------|--------|------|
| 搜索栏显示 | 悬停时显示 / 始终显示 / 始终隐藏 | 始终显示 | 控制搜索栏的显隐行为 |
| 搜索栏位置 | 顶部 / 中上 / 居中 / 中下 / 底部 | 居中 | 搜索栏在垂直方向上的位置 |
| 搜索栏圆角 | 胶囊(偏圆) / 圆角 / 直角 | 胶囊 | 搜索栏外框的圆角大小 |
| 壁纸遮罩 | 滑块 0~0.6 | 0(无遮罩) | 在壁纸之上叠加半透明黑色层，提升白色文字可读性 |
| 图标透明度 | 滑块 0~1 | 0.45 | 控制角落按钮和搜索引擎图标的透明度 |
| 搜索引擎 | Google / Bing / Baidu / DuckDuckGo | Google | 浏览器扩展模式下此选项隐藏，强制使用浏览器默认搜索引擎 |

底部的「恢复默认设置」按钮将所有界面设置恢复出厂值——搜索栏始终显示、居中位置、胶囊圆角、无遮罩、图标透明度 0.45。重点区分：**网页版**会将搜索引擎也恢复为 Google；**扩展版**不恢复搜索引擎（因为扩展版强制使用浏览器自带搜索引擎，不显示搜索引擎选项，恢复为 Google 反而会导致图标与实际搜索行为不一致）。

#### 壁纸分页

列出五种壁纸来源，每项显示名称和简短描述，当前选中的高亮。点击某个来源：
- 切换到新来源，更新高亮显示
- 重新渲染该来源的配置区域（不关闭模态窗口）
- 刷新一级面板
- Bing 来源立即加载壁纸；其他来源由用户在配置区域操作后生效

每个来源下方有专属配置区：
- **Bing：** 无需额外配置，提示「根据当前语言自动选择 Bing 市场区域」
- **本地上传：** 提示在一级面板中点击「＋」上传
- **文件夹：** 「选择文件夹」按钮 + 轮换策略单选（随机/顺序）
- **RSS：** Feed URL 输入框 + 「拉取图片」按钮 + 状态提示
- **API：** API URL 输入框 + JSON 图片路径输入框 + 「保存配置」按钮

#### 快捷键分页

三项设置：

- **命令面板快捷键：** 点击输入框后按下组合键录制。默认 Ctrl+K
- **隐藏面板快捷键：** 同上。默认 Ctrl+Shift+K
- **显示推荐：** 复选框，控制命令面板是否显示「高频推荐」区域。默认开启

录制规则：必须包含 Ctrl 键 + 一个字母键（A-Z），可选 Shift。不允许与另一个面板的快捷键相同。

#### 关于分页

显示 PlainTab 名称、版本号（v3.2.0）、简短介绍、GitHub 链接、作者和许可证信息。纯展示，无交互控件。

## 约束清单

### 一级面板

- 点击齿轮按钮打开/关闭（切换），已打开时再次点击不会重复打开
- 打开时齿轮按钮旋转 60 度，关闭时恢复
- 与语言面板互斥：打开一个自动关闭另一个
- 模态窗口打开时一级面板自动关闭
- 点击面板外区域 → 关闭
- 按 Escape → 关闭
- 面板宽度固定（280px），不随内容变化

**画廊（本地上传模式）：**
- 所有格子统一 16:9 比例，圆角统一
- 列宽固定像素值，1 张=1 列、2 张=2 列、3 张=3 列、4 张=2 列、5+ 张=3 列
- 缩略图缓存缺失 → 从原图即时生成 → 补写缓存
- 原图也丢失 → 该格子显示为空，不影响其他格子
- 图片上限 12 张，满额后「＋」按钮不显示
- 批量上传串行处理，超过剩余空位的自动跳过
- 文件选择器取消时不执行任何操作
- 上传去重：同名且同大小跳过
- 保存中途崩溃 → 最多留下孤立原图，安全忽略

**保存顺序（崩溃安全）：**
1. 原图先存入大文件存储
2. 再将 ID 加入轮换列表
3. 存缩略图
4. 存元数据
**删除顺序：**
1. 先从轮换列表移除 ID（断引用）
2. 删缩略图
3. 删元数据
4. 最后从大文件存储删原图

**拖拽排序：**
- 仅 2 张及以上启用
- 长按 300ms 触发，移动超 6px 取消
- 触屏设备禁用页面滚动防冲突
- 删除按钮区域长按不触发拖拽
- 被拖拽卡片移到页面顶层跟随指针；原位虚线占位
- 松手时落点高亮，200ms 滑入动画
- 松手在外部 → 回原位，不保存

**资源释放：**
- 面板关闭时释放此期间创建的所有临时图片资源
- 切换源或删除最后一张后切回 Bing 时也释放

### 模态窗口

- 快捷键 Ctrl+Shift+W 打开（可在快捷键分页中修改，但此处为固定默认值）
- 一级面板的「高级设置」按钮也可打开
- 已打开时再次触发不会重复打开
- 按 Escape 或点击遮罩区域 → 关闭
- 点击窗口内部空白区域不会关闭
- 窗口居中浮动，小屏幕（宽不足 720px）时自适应缩小
- 左侧分页切换有平滑过渡，不关闭窗口
- 所有设置修改即时保存，无需手动确认

### 壁纸来源切换

- 切换来源时，保留旧来源的配置（如 RSS URL、API URL），方便切回来时恢复
- 点击来源后不关闭模态窗口，只更新高亮和配置区域
- Bing 来源切换后立即加载壁纸；其他来源由用户在配置区域操作后生效
- 文件夹来源若选择失败（取消或空文件夹），保持当前来源不变
- RSS/API 的 URL 需校验为合法 http/https 地址

### 设置持久化

- 所有设置项在修改时即时保存，下次打开新标签页自动恢复
- 「恢复默认设置」仅影响界面分页的选项，不涉及壁纸数据、语言、快捷键配置、快捷链接数据
- 扩展版恢复默认时不改动搜索引擎（扩展版无搜索引擎选择，恢复为 Google 会导致图标与搜索行为不一致）
- 网页版恢复默认时将搜索引擎一并恢复为 Google
- 扩展模式下「搜索引擎」行完全隐藏

### 快捷键录制

- 必须包含 Ctrl + 有且仅有一个字母键（A-Z），可含 Shift
- 纯功能键（如 F1、方向键）无效
- 录制中按 Escape 或点击框外 → 取消，恢复原值
- Normal 和 Hidden 面板快捷键不允许相同 → 录制时拒绝并提示冲突
