# 全局交互需求规格

## 行为概述

全局交互系统管理新标签页上所有的键盘快捷键、鼠标行为和面板之间的协调。它是整个页面的「控制中枢」——决定什么操作在什么时候被触发。

### 页面启动流程

每次打开新标签页，以下事件依次发生：

1. 页面结构开始加载
2. **壁纸立即显示**——在浏览器还没画第一帧之前，一张压缩版壁纸就被写入屏幕。用户看到的第一帧就是有壁纸的，零白屏
3. 浏览器首次绘制完成
4. 其余界面元素加载（搜索栏、按钮、面板等）
5. 语言翻译表加载
6. 壁纸来源引擎加载
7. 主程序启动：检测运行环境 → 恢复用户设置 → 加载高清壁纸 → 绑定所有交互事件 → 页面就绪

### 运行环境检测

启动时自动判断当前是浏览器扩展模式还是普通网页模式。判定依据是浏览器扩展特有 API 是否存在且有效。

- **扩展模式**：搜索引擎选择器隐藏，搜索使用浏览器内置搜索引擎，图标变为静态放大镜
- **网页模式**：完整的搜索引擎选择器，点击引擎图标可循环切换

### 键盘快捷键

所有快捷键通过一个全局按键监听器处理，按优先级分发：

1. **最高优先：模态窗口**。打开时 Escape 关闭模态窗口，其他按键正常处理
2. **次优先：命令面板**。打开时所有按键交给命令面板处理（导航、回车、退出等）
3. **无面板打开时：**

| 快捷键 | 行为 |
|--------|------|
| Escape | 关闭所有面板，隐藏右上角按钮 |
| Ctrl+K | 打开普通命令面板 |
| Ctrl+Shift+K | 打开隐藏命令面板 |
| Ctrl+Shift+W | 打开设置模态窗口 |
| Enter（搜索框聚焦时） | 执行搜索 |

普通面板和隐藏面板互斥：一个打开时按另一个的快捷键不切换，而是提示「当前 XX 模式 — 请按 Esc 关闭后再试」，提示 2 秒后自动消失。

### 鼠标行为

**右上角热区：** 鼠标移到页面右上角（约 180×130 像素范围）时，右上角的设置齿轮按钮和语言地球按钮淡入显示。鼠标移开 400 毫秒后自动隐藏。但如果设置面板或语言面板开着，按钮保持显示不隐藏。

**屏幕中心区域：** 仅当搜索栏显示模式设为「悬停时显示」时生效。鼠标进入屏幕中央区域（宽 30%-70%，高 42%-58%）时搜索栏出现，移开 150 毫秒后消失。如果正在搜索框中打字，搜索栏无论如何不消失。

**点击壁纸区域：** 点击页面任何非 UI 控件的壁纸区域 → 搜索框立即获得焦点，用户可以直接开始打字搜索。这是最自然的搜索入口——看到壁纸，点一下，搜。点击面板外的区域 → 关闭面板。点击命令面板的半透明遮罩 → 关闭命令面板。

### 面板协调

四类面板共存时的层级和互斥规则：

| 面板 | 打开方式 | 互斥关系 |
|------|---------|---------|
| 一级设置面板 | 点击齿轮按钮 | 与语言面板互斥 |
| 语言面板 | 点击地球按钮 | 与一级设置面板互斥 |
| 设置模态窗口 | Ctrl+Shift+W 或一级面板的「高级设置」 | 打开时关闭一级面板 |
| 命令面板 | Ctrl+K / Ctrl+Shift+K | 独立覆盖层，不与其他面板互斥 |

同一面板不能重复打开——已打开时再次触发不会做任何事。关闭函数是幂等的——对已经关闭的面板再关一次也安全。

### 角落按钮动效

**齿轮按钮：** 默认半透明（透明度可调），鼠标悬停时变为完全不透明并旋转 30 度，按下时缩小到 92%。面板打开时完全不透明并旋转 60 度。

**语言按钮：** 默认完全隐藏，进入热区时出现。悬停时略微放大，按下时缩小。

### 窗口尺寸响应

窗口宽度小于 480px 时：按钮缩小、面板缩小、搜索栏变宽以适应小屏幕。

## 约束清单

### 键盘

- 快速连续按 Ctrl+K → 只打开一次，不会重复弹出
- 同时按 Ctrl+K 和 Ctrl+Shift+K → 只会触发一个（键盘事件串行）
- 命令面板动画未播完时按 Escape → 关闭动画覆盖打开动画
- 页面失去焦点时键盘事件不响应

### 鼠标

- 鼠标热区的检测使用像素坐标，窗口缩小时自动适应
- 右上角按钮的 400ms 隐藏延迟用于防止鼠标快速路过时闪烁
- 面板打开期间角落按钮始终保持可见（清除隐藏延迟）
- 搜索栏 150ms 隐藏延迟用于防止鼠标短暂移出时闪烁

### 面板

- 打开命令面板不主动关闭其他面板（命令面板覆盖在上面）
- 打开模态窗口时自动关闭一级面板
- 关闭命令面板后保持其他面板原始状态不变
- 点击面板上的空白区域被拦截，不会触发「点击面板外区域关闭」

### 面板并发场景

| 场景 | 行为 |
|------|------|
| 命令面板打开时按 Ctrl+Shift+W | 模态窗口在命令面板下层打开，两者独立操作 |
| 模态窗口打开时按齿轮按钮 | 关闭模态窗口，打开一级面板 |
| 模态窗口 + 命令面板同时打开 | 模态窗口在下层，命令面板在上层，各自独立 |
| 快速连续按 Ctrl+Shift+W | 模态窗口幂等——不重复打开 |
