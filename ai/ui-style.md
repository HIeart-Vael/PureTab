# UI 风格规则

## 理念（Philosophy）

* 以极简主义为第一原则
* 内容优先于装饰
* 不制造视觉噪音

---

## 布局（Layout）

* 单一用途界面（一个区域只做一件事）
* 避免过深的嵌套结构
* 保持清晰且浅层的层级结构

---

## 动效（Animation）

* 仅允许轻微、克制的动画
* 仅使用 opacity（透明度）和 transform（变换）
* 禁止触发布局重排（layout-triggering）的动画
* 避免过度运动与复杂动效

---

## 视觉系统（Visual System）

* 仅在必要时使用柔和阴影
* 模糊效果（blur）需谨慎使用
* 禁止重度玻璃拟态（glassmorphism）
* 保持整体视觉“安静感”和稳定性

---

## 交互（Interaction）

* 用户输入必须立即响应
* UI 反馈不能有延迟
* hover 状态必须克制、轻微、不干扰内容
